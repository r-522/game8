# マスターデータ仕様書 — 東京Bro（喧嘩番長-Crash Battle-）

## 技マスターテーブル（move_master）

### テーブル定義

| カラム名 | 型 | 説明 |
|---------|-----|------|
| id | int（PK） | 技ID（一意） |
| move_name | string | 技名称 |
| move_type | enum | 技タイプ（NORMAL / SUPER / MENCHI / TANKA） |
| base_power | int | 基本威力（ダメージ倍率の基底値、0=ダメージなし） |
| cost_mp | int | 消費ゲージ量（メンチゲージ/タンカゲージ） |
| acquire_condition | string | 習得条件（初期装備 / キャラ進化後 / スキルLv到達） |
| effect_type | enum | 効果タイプ（DAMAGE / HEAL / BUFF / DEBUFF / ALL） |
| description | string | ゲーム内説明テキスト |

## 敵マスターテーブル（enemy_master）

### テーブル定義

| カラム名 | 型 | 説明 |
|---------|-----|------|
| id | int（PK） | 敵ID |
| name | string | 敵名称 |
| enemy_type | enum | 敵タイプ（NORMAL / ELITE / BOSS） |
| hp | int | 最大HP |
| attack_power | int | 攻撃力 |
| defense_power | int | 防御力 |
| otokogi_value | int | 撃破時の男気変動値（正=男気UP、負=男気DOWN） |
| appear_stage_ids | string | 出現ステージIDリスト（カンマ区切り） |
| drop_item_ids | string | ドロップアイテムIDリスト |
| trait | enum | キャラ特性（NONE / PENETRATE / REFLECT / EXPLODE） |
