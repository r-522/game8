# マスターデータ仕様書 — 喧嘩番長6 Soul & Blood

## 技マスターテーブル

### テーブル定義

| カラム名 | データ型 | 説明 |
|---------|---------|------|
| id | INT (PK) | 技ID（一意） |
| name | VARCHAR(64) | 技名（日本語） |
| type | ENUM | 技タイプ（WEAK/STRONG/GRAB/KIAI/SUPER_KIAI/SPECIAL） |
| power | INT | 基礎威力（ATKパラメータへの倍率×100） |
| sp_cost | INT | 消費気合ゲージ量（通常攻撃は0） |
| unlock_condition | VARCHAR(256) | 習得条件（ボス撃破/ゲーム開始時/購入等） |
| shibu_penalty | BOOLEAN | シブ度低下フラグ（1=低下あり） |
| hit_frames | VARCHAR(32) | ヒットボックス有効フレーム範囲（例："8-12"） |
| combo_position | ENUM | コンボ内での位置（OPENER/MIDDLE/FINISHER/STANDALONE） |
| description | VARCHAR(256) | 技の説明文 |

### テーブル定義

| カラム名 | データ型 | 説明 |
|---------|---------|------|
| id | INT (PK) | アイテムID（一意） |
| name | VARCHAR(64) | アイテム名 |
| category | ENUM | カテゴリ（HEAL/BOOST/KEY/WEAPON/FACILITY） |
| effect_type | ENUM | 効果タイプ（HEAL_HP/HEAL_SP/ATK_UP/DEF_UP/STATUS等） |
| effect_value | INT | 効果量（HP回復量、ステータス上昇値等） |
| effect_duration | INT | 効果継続時間（フレーム数、0は永続/即時） |
| price | INT | 購入価格（円）、0は購入不可 |
| max_stack | INT | 所持上限数 |
| description | VARCHAR(256) | アイテム説明文 |

## ステージマスターテーブル
