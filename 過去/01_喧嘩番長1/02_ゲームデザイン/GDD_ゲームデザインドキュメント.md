# GDD — 喧嘩番長 ゲームデザインドキュメント

## ゲームループ概要

本作の基本ゲームループは以下の流れを繰り返す構造となっている。

```
[フィールド探索] → [遭遇/イベント発生]
       ↓
[メンチ（視線対決）] → 逃げる or 受ける
       ↓（受ける）
[タンカ（啖呵バトル）] → 成功/失敗
       ↓
[戦闘（どつき合い）] → 勝利/敗北
       ↓
[男気更新・報酬取得] → [フィールドへ戻る]
```

フィールドでは自由に各エリアを歩き回り、不良との遭遇、一般市民からの依頼、ヒロインとの会話など多彩なイベントが発生する。進行はストーリー進行に必要なターゲット（各学校の番長）を倒すことで次章が解放される線形+オープン構造。

---

## メンチシステム詳細仕様

### 概要
フィールドでRボタンを押すと「メンチモード」に入る。目から「メンチビーム」が放たれ、ロックオンした相手のビームとぶつかり合う視線対決が発生する。

### 操作
- Rボタン長押し: メンチビーム発射・維持
- 対象に照準を合わせ続けることで「メンチ成立」判定

### メンチビーム成立条件
- 相手がメンチを返してきた後、○ボタンを押す → ケンカ成立 → タンカへ
- 相手がひるんだ場合 → 一方的なメンチ成立 → 有利状態で戦闘開始

### 気合消費
- メンチビームは気合ゲージを消費する
- 気合が不足しているとメンチが弱くなる

### 男気ペナルティ
- メンチを切らずに攻撃する → 男気低下（「ちょっかい」扱い）
- 相手のメンチを無視する → 男気低下
- 男気が「シャバゾウ」状態の場合、メンチビームが相手に届かなくなる

## タンカシステム詳細仕様

### 概要
メンチビーム成立後の威圧合戦。画面上にセリフ（タンカ）の断片がバラバラに表示され、対応するボタンを制限時間内に正確に入力することで「啖呵を切る」ことができる。

### 入力仕様
- 制限時間: 約5〜8秒（敵の強さによって変動）
- 画面表示: セリフの各単語にボタンアイコンが対応
- 入力順序: 表示されたボタンを順番通りに押す

### 成功・失敗の効果
| 結果 | 効果 |
|------|------|
| タンカ成功（正解） | 先制攻撃ボーナス（一定時間攻撃力上昇）、男気上昇 |
| 裏正解入力 | 強化版先制攻撃 or 特殊アイテム入手 |
| タンカ失敗 | 敵に先制攻撃を許す、男気低下 |

### 裏正解システム
- 一部のタンカには「正解ルート」以外に「裏正解ルート」が存在
- 裏正解は通常の正解より難しい入力が必要
- 成功すると通常より大きな先制攻撃ボーナスや特殊アイテムを獲得

## 戦闘システム詳細仕様

### 基本操作
| ボタン | 動作 |
|--------|------|
| □ | 弱攻撃 |
| △ | 強攻撃 |
| L1 | 投げ・つかみ |
| ○ | ジャンプ / 回避 |
| × | ガード |
| R2 | 怒声（敵をひるませる） |
| L2+△など | 特殊技入力 |

### コンボシステム
- 弱攻撃・強攻撃の2ボタンを組み合わせたコンボを繰り出す
- 連続ヒットでコンボカウンターが増加、ダメージ補正が上昇
- 代表コンボ例: □□□△（弱弱弱強）→ 3ヒットから打ち上げ

### 必殺技・超必殺技
- 自宅メニューで事前に技を「装備」してから使用する仕様
- 気合ゲージを消費して発動
- 超必殺技: 気合を大量消費、高威力・長い演出あり
- 180種類以上の技が存在し、番長度（レベル）上昇で新技が解放される

### ガードシステム
- ×ボタンでガード。完全防御ではなく一定割合のダメージを軽減
- ガード崩し技あり（強攻撃の一部がガード不能）

### うんこ座りシステム
- フィールド・戦闘中にその場で「うんこ座り」をすると気合ゲージが回復
- ただしうんこ座り中は体力が微減するリスクあり

### 仲間呼び出し
- 男気が高い（シブイ）状態のみ電話で仲間（舎弟）を呼び出して一緒に戦わせることが可能
- 仲間はAI制御で自律的に戦う

### 武器（凶器）
- 街中に落ちているバット、鉄パイプ、椅子などを拾って使用可能
- 使用すると男気が低下するため多用は禁物

## 男気システム詳細仕様

### 状態区分

| 状態 | 条件 | 効果 |
|------|------|------|
| シブイ（最高） | 男気高値維持 | 全施設利用可・仲間呼び出し可能 |
| 普通 | 中程度 | 基本機能使用可 |
| シャバゾウ | 男気低下 | 施設利用不可・メンチビーム無効・仲間呼び出し不可 |
| シャバ王 | 極度の低下 | 特定条件を満たさない限り男気回復不可 |

### 男気上昇アクション
- 喧嘩勝利
- タンカ成功
- 一般市民の依頼達成（人助け）
- シブイ行動全般（義理・人情に基づく行動）

### 男気低下アクション
- メンチなし攻撃（ちょっかい）
- 相手のメンチを無視
- タンカ失敗
- 喧嘩から逃げる
- 武器（凶器）使用
- 一般人・女子生徒への暴力
- 車を殴る
- 悪い金の収受（戦意のない相手や一般人が落とした金を拾う）

## 成長・パラメータシステム

### 喧嘩慣れ度・番長度
- 戦闘を重ねるごとに「喧嘩慣れ度」が蓄積
- 一定値に達すると「番長度」が上昇
- 番長度上昇による恩恵:
  - 能力値（体力上限、攻撃力、防御力）向上
  - 新技の習得が可能になる

### 主要パラメータ
- **体力**: ダメージで減少、ゼロでゲームオーバー
- **スタミナ**: 行動で減少、低下すると体力の上限が下がる
- **気合**: 技使用・メンチビーム・怒声で消費、うんこ座りで回復
- **男気**: 行動によって増減する倫理的評価値

## マップ・フィールドシステム

### 極東線システム
- プレイヤーは「極東線」各駅を電車で移動できる
- 各駅周辺に対応する学校・商店街エリアが存在
- 電車移動中は時間が経過し、時間帯によって出現するイベントが変わる

### 時間帯
- 昼・夕方・夜の3時間帯
- 時間帯によって不良の出現率、店の営業状況が変化

### 施設一覧
| 施設 | 機能 | 利用条件 |
|------|------|---------|
| コンビニ | 回復アイテム購入 | 常時 |
| テーラー | 服の購入・変更 | 男気条件あり |
| 床屋 | 髪型変更 | 男気条件あり |
| 雑貨屋 | 各種アイテム購入 | 男気条件あり |
| 自宅（俺様の部屋） | セーブ・装備変更・体力回復 | 常時 |

## その他システム

### セーブシステム
自宅「俺様の部屋」でのみセーブ可能。ゲームオーバー時はオープニングまで戻される高難易度設計。

### ファッションシステム
テーラーで特攻服や通常ファッションを購入・変更。特攻服はポケット数が多くアイテム所持数が増えるが、目立つため特定イベントに影響する場合あり。

### 伝説達成システム
特定条件（特定の技で敵を倒す、メンチで無双するなど）を達成すると「伝説アルバム」に記録される実績システム。全達成でコンプリートボーナスあり。

### 恋愛システム
恋愛イベントの交際相手として選択可能なヒロインは浅岡早紀・中西真奈美の2名。佐々木レイナはスケ番編の主人公であり、恋愛対象ではない。会話選択肢を積み重ねて関係を深め、最終的にカップル成立を目指す。フラれる可能性もあり、好感度管理が必要。
