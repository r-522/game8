# 画面仕様書 — 喧嘩番長6 Soul & Blood

---

## 画面1：タイトル画面

### 概要
ゲーム起動後に表示される最初の画面。「喧嘩番長6〜ソウル＆ブラッド〜」のロゴが上画面に表示され、背景では朝比奈大吾と片桐龍治が対峙するシルエット演出が流れる。下画面には「PRESS START」のメッセージが点滅表示。

### 表示要素
- **上画面：** ゲームタイトルロゴ（日本語＋英語）、背景アニメーション（キャラクターシルエット対峙）、立体視（3D）対応
- **下画面：** 「PRESS START BUTTON」テキスト（点滅）、コピーライト表記、バージョン情報
- **BGM：** タイトルテーマ（昭和風ロック調、ループ）
- **SE：** ロゴ出現時のエフェクト音

### 操作方法
| 入力 | 動作 |
|------|------|
| STARTボタン | メインメニューへ遷移 |
| Aボタン | メインメニューへ遷移 |
| しばらく放置 | デモムービー（オープニングデモ）が自動再生される |

### 入力受付
- タイトル画面表示から0.5秒後に入力受付開始
- デモムービー中はSTARTで中断可能

---

## 画面2：メインメニュー

### 概要
タイトル画面からSTART押下後に表示されるメニュー選択画面。ゲームの各モードへアクセスする。

### 表示要素
- **上画面：** メインビジュアル（大吾の立ち絵または九鬼島高校外観）
- **下画面：** メニュー項目リスト（縦並び）
  - はじめから
  - つづきから
  - みんなで戦う（全国ランキングバトル）
  - せってい
- 選択中の項目はハイライト表示、大吾の声でコメント
- **BGM：** タイトルテーマ継続

### 操作方法
| 入力 | 動作 |
|------|------|
| 十字キー上下 | カーソル移動 |
| Aボタン | 選択確定 |
| Bボタン | タイトル画面へ戻る |

### 入力受付
- 全項目常時受付
- Wi-Fi未接続時は「みんなで戦う」を選択するとWi-Fi設定案内を表示

## 画面3：フィールド画面（メインゲームプレイ）

### 概要
本作のメイン画面。3Dで構成された華ヶ崎市内のフィールドを自由に移動して探索する。敵キャラクターがフィールドを徘徊しており、主人公に気づくと追跡してくる。

### 表示要素
- **上画面（メイン）：** 3Dフィールドビュー（三人称視点）、キャラクターモデル、3D立体視対応
- **下画面（サブ）：** タッチスクリーンにミニマップ（周辺エリア表示）、HPバー、気合ゲージ、シブシャバ度インジケーター
- **上画面HUD（オーバーレイ）：** 残り時間表示（学期イベント期限）、エリア名称表示（入場時）、マーカー（ストーリー目標地点）
- **敵アイコン：** ミニマップ上に赤点で敵位置表示

### 操作方法
| 入力 | 動作 |
|------|------|
| スライドパッド | キャラクター移動 |
| Rボタン長押し | メンチモード切替 |
| Aボタン | アクション（ドア開閉、NPCに話す、アイテム拾い） |
| STARTボタン | ポーズメニュー開く |
| 十字キー左右 | カメラ水平回転 |
| 十字キー上下 | カメラ垂直角度調整 |
| Yボタン | 走る（ダッシュ） |

### 入力受付
- イベントシーン中は移動操作を無効化
- ショップ・アジト入室時はフィールド操作を無効化

## 画面4：メンチ画面（メンチビームモード）

### 概要
Rボタン長押しで発動するメンチモード。フィールドに重ねてメンチ照準UIが表示され、主人公の目からビームが伸びる演出が加わる。

### 表示要素
- **上画面：** 3Dフィールドビュー（同一）＋主人公目線の照準ガイド（クロスヘア）、ビームエフェクト（赤い光線）
- **下画面：** ターゲット情報（敵名前・ランク・体力バー）、メンチ成功率ゲージ、近距離の敵一覧サブ表示
- **演出：** 主人公の目が細まり鋭くなる顔アップアニメーション（初発動時）

### 操作方法
| 入力 | 動作 |
|------|------|
| Rボタン長押し | メンチモード維持 |
| スライドパッド | 照準移動 |
| Rボタン離す | ビーム発射（照準をロックした方向へ） |
| Bボタン | メンチモードキャンセル（フィールドへ戻る） |

### 入力受付
- 敵が照準エリア内にいる場合のみビーム発射が有効
- ロックオン成立後に離すとビーム発射、ロックオン前に離すと空振り

## 画面5：タンカバトル画面

### 概要
メンチビーム成立後に移行するセリフバトル画面。スライドパッドの方向入力で主人公が威勢の良い口上（タンカ）を繰り出す。

### 表示要素
- **上画面：** 大吾と敵キャラクターのアニメーション（フルボイス台詞）、「タンカバトル」ロゴ演出
- **下画面：** 方向入力ガイド（矢印表示）、制限時間バー、連続成功数カウンター、シャミセンヒント表示（残り2回以下の時）
- **タン語選択：** 最終入力後に4択セリフ選択ウィンドウ表示

### 操作方法
| 入力 | 動作 |
|------|------|
| スライドパッド（指示方向） | タンカ入力成功 |
| スライドパッド（逆方向） | タンカ入力失敗 |
| Yボタン | 不正行為（ズル入力、シブ度低下） |
| L+Rボタン | シャミセン発動 |
| Aボタン（タン語選択時） | タン語決定 |

### 入力受付
- 各タンカは1.5秒〜2秒の入力受付時間
- 制限時間切れは失敗判定
- 全タンカ成功時は「先制攻撃ボーナス」付与（戦闘開始時に相手が硬直）

## 画面6：3D戦闘画面

### 概要
本作のメインアクション画面。タンカバトル後、または敵との直接衝突で開始される3Dアクション格闘画面。上画面でアクション、下画面でステータス管理を行う。

### 表示要素
- **上画面：** 3D格闘フィールド（専用バトルステージ）、キャラクターモデル（プレイヤーと敵）、ヒットエフェクト、超気合技演出カットイン、立体視対応
- **下画面：** HPバー（自分・敵）、気合ゲージ、コンボカウンター、タイムカウント（一部ボス戦）、ミニ操作ガイド
- **カットイン演出：** 超気合技発動時にキャラクター顔アップのカットイン画面（下画面使用）

### 操作方法
| 入力 | 動作 |
|------|------|
| スライドパッド | 移動 |
| Y | 弱攻撃（最大4連続） |
| X | 強攻撃（最大3連続） |
| A | 掴み・投げ・ダウン乗り |
| L | ガード |
| L+スライドパッド | ステップ回避 |
| X長押し | チャージ攻撃（ガード崩し） |
| X+A | 気合技/超気合技 |

### 入力受付
- 全操作は戦闘中常時受付
- ヒットストップ中（攻撃ヒット時の停止演出中）は操作入力を一時バッファリング
- 超気合技発動演出中はカットイン終了まで通常操作無効

---

## 画面7：ステータス確認画面

### 概要
ポーズメニューからアクセスするキャラクター情報画面。パラメータ・習得技・称号・シブシャバ度などを確認できる。

### 表示要素
- **上画面：** キャラクター立ち絵（現在の服装・怪我状態を反映）、称号表示、シブシャバ度メーター（ビジュアル表示）
- **下画面：** タブ切替式の情報パネル
  - 「パラメータ」タブ：5パラメータの現在値・上限値（バー表示）、総喧嘩魂・レベル
  - 「技」タブ：習得済み技一覧（気合技・超気合技）、コンボカスタマイズリンク
  - 「称号」タブ：獲得した称号一覧、現在装備中の称号
  - 「記録」タブ：倒した敵数・収集校章数・治安データ

### 操作方法
| 入力 | 動作 |
|------|------|
| 十字キー左右 | タブ切替 |
| 十字キー上下 | リスト内スクロール |
| Aボタン | 詳細表示（技の詳細説明など） |
| Bボタン | ポーズメニューへ戻る |
| Rボタン | ページめくり（リストが長い場合） |

### 入力受付
- 読み取り専用（ここでのパラメータ変更はできない）
- コンボカスタマイズへはAボタンで専用画面へ遷移
