# GDD — 喧嘩番長5 漢の法則

## ゲームループ概要

本作のゲームループは「探索→遭遇→メンチ→タンカ→バトル→結果→成長」の7フェーズサイクルで構成される。

```
[フィールド探索]
      ↓
[敵と遭遇（視界内）]
      ↓
[メンチ（視線対決）]
      ↓ 成立
[タンカ（口頭挑発バトル）]
      ↓
[喧嘩バトル（アクション入力）]
      ↓
[勝利：ダチる or 撃退]
      ↓
[男気ゲージ更新・経験値獲得・縄張り変化]
      ↓
[フィールド探索に戻る]
```

ゲーム全体の大目標は「阿弥浜沿線全域の縄張りを制覇し、8勢力の番長・ヘッド全員とダチになる」こと。ダチになった仲間はアジトに集まり、集団喧嘩の際に呼び出せる。

---

## メンチシステム詳細仕様

### 概要
フィールド上で敵キャラクターの視界範囲内に入ったとき、プレイヤーがRボタンを押すと「メンチビーム」が発動する。相手も同様にメンチを返した場合、○ボタンを押すことで正式なケンカ申込みが成立する。

### メンチ顔選択
メンチビームを発動した後、以下の顔の種類を十字キーで選択できる。

| 顔の種類 | 効果 |
|---|---|
| 漢（おとこ） | 標準。相手の台詞がリスペクト系になりやすい |
| 嘲（あざ） | 相手を侮辱。タンカが激化する傾向 |
| 怒（いかり） | 威圧感を高める。相手がひるむ場合あり |
| 笑（わら） | 余裕を示す。場の雰囲気を和らげる効果 |
| 凄（すご） | 重圧を与える。男気ゲージ高いと効果大 |

### ルール違反
- メンチなしで先に殴りかかると「シャバ行為」と判定
- シャバ行為は男気ゲージを大幅に低下させる
- シャバ判定が続くとモブキャラから「シャバい奴」と侮辱されるようになる

### 複数人・集団遭遇
- 3人以上のグループとメンチを切ると集団喧嘩モードへ移行
- 集団喧嘩では事前に電話でマブダチを呼び出しておく必要がある

## タンカシステム詳細仕様

### 基本フロー
メンチ成立後、タンカバトル画面に移行する。画面上部からランダムな順序でセリフの断片（単語・フレーズ）が落下してくる。対応するボタン（○△□×）を制限時間内に押してセリフを完成させる。

### 成功・失敗の影響
| 結果 | バトルへの影響 |
|---|---|
| タンカ完勝（全フレーズ完成） | 相手HP30%減少した状態でバトル開始、攻撃力+20% |
| タンカ勝利（半数以上完成） | 相手HP15%減少、攻撃力+10% |
| タンカ引き分け | HP増減なし、通常ステータスでバトル開始 |
| タンカ敗北 | 自分のHP15%減少、防御力-10% |
| タンカ完敗 | 自分のHP30%減少、移動速度-15% |

### シャミセン（L+R同時押し）
通常のタンカとは別に、L・Rボタン同時押しで「シャミセン」を発動できる。単語を自分で組み合わせてセリフを作るシステムで、成功した場合の効果は通常タンカより大きいが、失敗した場合のペナルティも大きい。

### ボス専用タンカ
各勢力のボスキャラには専用タンカ台本が用意されており、通常の断片入力とは異なる特殊演出が挟まる。成功した場合の特殊カットインが再生される。

## 戦闘システム詳細仕様

### 通常攻撃入力
画面上から落下してくる攻撃アイコン（○△□×およびL/R）と同じボタンを、アイコンが判定ラインに重なった瞬間に押す。判定は「PERFECT・GREAT・GOOD・BAD・MISS」の5段階。PERFECTが連続するとコンボボーナスが積み重なる。

### コンボシステム
技は「コンボ重視型」「大技狙い型」「アクロバティック型」など複数のカスタムデッキからプレイヤーが事前に構成できる。コンボ継続中は攻撃力倍率が上昇し、10コンボ以上で「乗り気」エフェクトが発動して演出が派手になる。

### 超気合技（ちょうきあいわざ）
各キャラクター固有の必殺技。気合ゲージが最大まで溜まった状態で特殊入力シーケンスを成功させると発動する。

| キャラクター | 超気合技名 | 効果 |
|---|---|---|
| 成瀬直人 | ジョー・クラッシュ | 連続猛打・高威力 |
| 高津昇吾 | ツルメソウル | 周囲全員に範囲攻撃 |
| 八神真 | 天才の一閃 | 一撃大ダメージ |
| 速水勇太（OB） | アメリカン☆ドリーム | 豪快な飛び蹴り連打 |

超気合技のレベルはイメージトレーニングで上げることができ、Lv1→Lv3まで威力と演出が強化される。

### ガード・回避
- 十字キー入力でステップ回避
- L長押しでガード（一定ダメージを無効化、ゲージ消費）
- タイミング良くガードするとカウンター発動

### ブチギレモード
敵から連続攻撃を受けると画面端に「ブチギレゲージ」が蓄積される。MAXになると自動でブチギレモードに移行し、一定時間ひるまずに猛攻撃が可能になる。ブチギレ中はBGMが変わり、特殊エフェクトが発動する。

### 集団喧嘩
主人公 + 電話で呼び出したマブダチ1名 + 張り番3名の計5人で戦う特殊モード。仲間はAIで自動行動するが、指示コマンドで「攻撃集中」「分散」「守れ」等を指示できる。集団喧嘩は主人公がやられた場合に仲間が代わりに戦い続けることも可能。

### 武器使用
拾った武器（鉄パイプ・コーンスタンド等）を使用すると攻撃力が上がる反面、男気ゲージが低下するリスクがある。武器の使用判断はプレイヤーの男気方針に委ねられている。

## 男気システム詳細仕様

### 概要
プレイヤーの「番長としての行動規範」を数値化したシステム。0〜100のゲージで表示される。

### ゲージ増加要因
- 正式なメンチから喧嘩を始める（+5）
- 相手が降参したときに手を差し伸べてダチる（+10）
- 弱い者を助ける（+15）
- タコさんの武勇伝を最後まで聞く（+8）
- 自分より格上の相手に挑む（+10）

### ゲージ減少要因（シャバ行為）
- メンチなしで先に殴りかかる（-15）
- 武器を使用する（-10）
- 一般人・子供への暴力（-30、大罰則）
- 敵が降参しているのに攻撃し続ける（-20）
- タンカを途中放棄する（-8）

### シャバ度パラメータ
男気ゲージとは別に「シャバ度」が存在する。シャバくなる技・行動を繰り返すと上昇し、閾値を超えると「シャバ神」ステータスになる。シャバ神状態ではモブキャラが軽蔑した態度を取り、タンカが不利になる等の社会的ペナルティが発生する。

### 男気ゲージの影響
| ゲージ値 | 称号 | 効果 |
|---|---|---|
| 80〜100 | 漢（おとこ） | タンカ難易度低下、相手が自発的に近づいてくる |
| 60〜79 | 硬派 | 標準状態 |
| 40〜59 | 普通 | 一部キャラから軽んじられる |
| 20〜39 | チンピラ | タンカ難易度上昇 |
| 0〜19 | シャバい奴 | モブから侮辱される、特定イベント発生不可 |

## 成長・パラメータシステム

### 主人公パラメータ
| パラメータ | 成長方法 |
|---|---|
| 体力（HP） | 喧嘩経験値・イメージトレーニング |
| 気合（MP） | イメージトレーニング・超気合技習得 |
| 攻撃力 | 技習得・アジト家具（筋トレ設備） |
| 防御力 | バトル経験・防御系アクセサリー |
| スピード | 走り込み系ミニゲーム |
| 男気 | 行動規範による変動 |
| 漢の器 | ダチ数に比例して上昇 |

### 技習得システム
- ゲームセンターで「喧嘩番長3 全国制覇アーケード」をプレイし、全国47都道府県の番長を倒すと各番長の超気合技が習得可能
- イメージトレーニングで既存技のレベルアップ
- 特定の武勇伝クリアで解放される技あり

## マップ・フィールドシステム

### エリア構成
阿弥浜沿線は5地区に分かれ、各地区は電車・徒歩でシームレスに移動可能。赤く表示されたエリアが未制覇の縄張りを示し、制覇するとプレイヤーカラーに変化する。

### 時間システム
ゲーム内時間は朝・昼・夕方・夜の4区分。時間帯によって出現するキャラクター・イベントが変化する。深夜は骸・K.M.Aなど危険な相手が増加する。

### ショップ・施設
- コンビニ：回復アイテム購入
- ゲームセンター：喧嘩番長3アーケード・各種ミニゲーム
- 餃子の王将：HP回復食事・多胡との交流
- ジーンズメイト：カスタマイズパーツ購入
- イメージトレーニング施設：各地区のパラメータ訓練

## その他システム

### ダチるシステム
敵を降参（白旗状態）にした後、Rトリガーを押すと「手を差し伸べる」演出が発生し友人化する。友人は100人以上作成可能。友人を増やすほど「漢の器」パラメータが上昇する。

### 子犬育成システム
鶴目高校周辺で捨てられた子犬を発見。好きな名前を付けて育てることができる。毎日エサをあげ・遊ぶことで好感度が上がる。好感度MAXになると喧嘩中もひるまず追いかけてきて、落ちているアイテムを拾って届けてくれる行動を取る。

### アジトリフォームシステム
入手したベンチ・学校の机・看板・バス停等の家具・オブジェクトをアジト内に自由配置できる。配置した家具の種類によってアジトに集まるダチのステータスが変化・強化される効果がある。

### 喧嘩鉄道99
番長を次々と倒していく連続挑戦モード。99番勝負で体力は回復なし（一部制限あり）。完走すると特別称号と専用超気合技が解放される。
