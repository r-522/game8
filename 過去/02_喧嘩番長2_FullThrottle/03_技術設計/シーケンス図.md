# シーケンス図 — 喧嘩番長2 Full Throttle

## シーケンス1: フィールドでの遭遇〜メンチ開始

```mermaid
sequenceDiagram
    participant Player as プレイヤー
    participant FieldSystem as FieldSystem
    participant Enemy as EnemyNPC
    participant MenchiSystem as MenchiSystem
    participant AudioManager as AudioManager
    participant OtokogiSystem as OtokogiSystem

    Player->>FieldSystem: R2ボタン押下
    FieldSystem->>FieldSystem: 近距離ヤンキーNPC探索（範囲チェック）
    FieldSystem->>Enemy: ロック対象確定
    Enemy->>FieldSystem: メンチ受諾（AI判定）

    FieldSystem->>MenchiSystem: startMenchi(player, enemy)
    MenchiSystem->>AudioManager: playSE("menchi_start")
    AudioManager-->>MenchiSystem: SE再生完了通知

    MenchiSystem->>Player: カメラを対決アングルに移動
    MenchiSystem->>Player: ビームエフェクト生成（両眼）
    MenchiSystem->>Enemy: ビームエフェクト生成（両眼）

    loop ビーム照射中
        Player->>MenchiSystem: R2長押し継続
        MenchiSystem->>MenchiSystem: lockProgress += dt × lockRate
        MenchiSystem->>Player: UI更新（ロック進捗表示）
    end

    MenchiSystem->>MenchiSystem: lockProgress >= threshold 確認
    MenchiSystem->>Player: タイミング判定リング表示（◎）
    Player->>MenchiSystem: ○ボタン（タイミング内）

    MenchiSystem->>MenchiSystem: 勝利判定 = true
    MenchiSystem->>AudioManager: playSE("menchi_win")
    MenchiSystem->>OtokogiSystem: addScore(+2, MENCHI_WIN)
    MenchiSystem->>FieldSystem: メンチ完了通知（勝利）
    FieldSystem->>FieldSystem: 次フェーズ=TANKA
```

## シーケンス2: メンチ勝利〜タンカ開始

```mermaid
sequenceDiagram
    participant FieldSystem as FieldSystem
    participant TankaSystem as TankaSystem
    participant DataManager as DataManager
    participant Player as プレイヤー
    participant Enemy as EnemyNPC
    participant AudioManager as AudioManager

    FieldSystem->>TankaSystem: startTanka(enemyId, menchiResult)
    TankaSystem->>DataManager: loadTanka(enemyId)
    DataManager-->>TankaSystem: TankaData（お題文・選択肢4個・正解index・裏正解index）

    TankaSystem->>AudioManager: playBGM("tanka_battle")
    TankaSystem->>Player: タンカ画面表示
    TankaSystem->>Enemy: 挑発アニメーション再生

    TankaSystem->>TankaSystem: タイマースタート（5.0秒）

    Note over TankaSystem: お題テキスト・選択肢を画面表示

    Player->>TankaSystem: 選択肢にカーソル移動（十字キー）
    TankaSystem->>Player: 選択ハイライト更新

    Player->>TankaSystem: ○ボタン（選択確定）

    alt 正解選択時
        TankaSystem->>TankaSystem: result = CORRECT
        TankaSystem->>AudioManager: playSE("tanka_correct")
        TankaSystem->>Player: 「正解！」エフェクト表示
        TankaSystem->>FieldSystem: タンカ完了（先制攻撃ボーナス付き）
    else 裏正解選択時
        TankaSystem->>TankaSystem: result = HIDDEN_CORRECT
        TankaSystem->>AudioManager: playSE("tanka_hidden")
        TankaSystem->>Enemy: 動揺アニメーション再生
        TankaSystem->>Player: 「ド正解！」スペシャルエフェクト表示
        TankaSystem->>FieldSystem: タンカ完了（超先制ボーナス付き）
    else 不正解・時間切れ
        TankaSystem->>TankaSystem: result = FAIL
        TankaSystem->>AudioManager: playSE("tanka_fail")
        TankaSystem->>FieldSystem: タンカ完了（ボーナスなし）
    end
```

## シーケンス3: タンカ結果〜戦闘開始

```mermaid
sequenceDiagram
    participant FieldSystem as FieldSystem
    participant CombatSystem as CombatSystem
    participant Player as プレイヤー
    participant Enemy as EnemyNPC
    participant AudioManager as AudioManager
    participant OtokogiSystem as OtokogiSystem

    FieldSystem->>CombatSystem: startCombat(player, enemy, tankaResult)

    alt tankaResult == HIDDEN_CORRECT
        CombatSystem->>Player: 超先制ボーナス適用（気合ゲージ30%事前チャージ）
        CombatSystem->>AudioManager: playSE("combat_super_start")
    else tankaResult == CORRECT
        CombatSystem->>Player: 先制攻撃ボーナス（最初の1撃ダメージ×1.5）
        CombatSystem->>AudioManager: playSE("combat_normal_start")
    else tankaResult == FAIL
        CombatSystem->>Enemy: 先手ボーナス（敵が先攻）
        CombatSystem->>AudioManager: playSE("combat_unfav_start")
    end

    CombatSystem->>AudioManager: playBGM("combat_theme_" + stageId)
    CombatSystem->>Player: 戦闘HUDをオーバーレイ表示（HP・気合ゲージ）
    CombatSystem->>Enemy: 戦闘AIをACTIVEに切替

    loop 戦闘ループ（60fps）
        CombatSystem->>Player: InputManager.getInput() → アクション実行
        CombatSystem->>Enemy: AI.selectAction() → 行動実行
        CombatSystem->>CombatSystem: HitDetector.checkCollisions()
        CombatSystem->>CombatSystem: applyDamage() if hit
        CombatSystem->>Player: HP・気合ゲージUI更新
    end

    alt プレイヤーHP > 0 && 敵HP == 0
        CombatSystem->>FieldSystem: 戦闘終了通知（PLAYER_WIN）
    else プレイヤーHP == 0
        CombatSystem->>FieldSystem: 戦闘終了通知（PLAYER_LOSE）
    end
```

## シーケンス4: 戦闘終了〜男気更新〜結果画面

```mermaid
sequenceDiagram
    participant CombatSystem as CombatSystem
    participant OtokogiSystem as OtokogiSystem
    participant GrowthSystem as GrowthSystem
    participant ResultScreen as 結果画面
    participant EventSystem as EventSystem
    participant Player as プレイヤー
    participant AudioManager as AudioManager

    CombatSystem->>OtokogiSystem: reportCombatResult(result, combatData)

    Note over OtokogiSystem: combatDataに武器使用有無・逃亡有無・正規フロー有無が含まれる

    alt 正規フロー（メンチ→タンカ→戦闘）かつ勝利
        OtokogiSystem->>OtokogiSystem: addScore(+5, LEGITIMATE_WIN)
    else 武器使用あり
        OtokogiSystem->>OtokogiSystem: subtractScore(-10, WEAPON_USED)
    end

    OtokogiSystem->>OtokogiSystem: checkLevelChange()
    OtokogiSystem->>Player: 男気ゲージUI更新

    CombatSystem->>GrowthSystem: reportVictory(enemyData)
    GrowthSystem->>GrowthSystem: kansExp += enemyData.expValue
    GrowthSystem->>GrowthSystem: checkLevelUp()

    alt レベルアップ発生
        GrowthSystem->>ResultScreen: レベルアップ通知
        ResultScreen->>Player: レベルアップ演出表示
        ResultScreen->>Player: 喧嘩魂ポイント振り分け画面表示
        Player->>ResultScreen: パラメータ振り分け入力
        ResultScreen->>GrowthSystem: applyPoints(assignments)
    end

    CombatSystem->>AudioManager: playBGM("result_fanfare")
    CombatSystem->>ResultScreen: showResult(player, enemy, combatStats)

    ResultScreen->>ResultScreen: 戦闘統計表示（ヒット数・ダメージ・時間）
    ResultScreen->>ResultScreen: 習得技チェック（新技解放があれば表示）

    ResultScreen->>EventSystem: checkPostCombatFlags(enemyId)

    alt ボス撃破後イベントあり
        EventSystem->>EventSystem: triggerEvent(bossDefeatedEventId)
        EventSystem->>ResultScreen: カットシーン再生要求
        ResultScreen->>Player: カットシーン開始
    else 通常戦闘終了
        ResultScreen->>Player: ○ボタン待機（確認入力）
        Player->>ResultScreen: ○ボタン押下
        ResultScreen->>CombatSystem: 結果画面終了
        CombatSystem->>FieldSystem: フィールドに戻る
    end
```
