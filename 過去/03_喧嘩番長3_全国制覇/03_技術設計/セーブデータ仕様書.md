# セーブデータ仕様書 — 喧嘩番長3 全国制覇

## セーブデータ構造定義

### フォーマット概要
- 保存形式：バイナリ形式（独自フォーマット）
- 保存先：PSPメモリースティックDuo（/PSP/SAVEDATA/ 以下）
- ファイル名：ULJM05302_XXXXXX（PSP標準セーブディレクトリ規約に準拠）
- チェックサム：CRC32（データ改ざん検知用）

## 保存タイミング

### 手動セーブ
- 旅館のセーブポイントから「セーブする」を選択した場合のみ保存
- フィールドや戦闘中にはセーブ不可（ポーズメニューからも不可）
- セーブ完了時にはフラッシュメモリへの書き込み完了確認後に「保存しました」表示

### 自動セーブ（オートセーブ）
- 本作には自動セーブ機能なし（PSP時代の標準仕様に従い手動のみ）
- 誤ってゲームを終了した場合は前回セーブ状態から再開

## ストレージ使用量

| 項目 | 容量 |
|------|------|
| セーブデータ1スロット | 約128バイト（実際の確保ブロック：128KB以下） |
| セーブスロット×3 | 約384バイト |
| PSPセーブディレクトリオーバーヘッド | 約4KB（ディレクトリ構造・サムネイル画像含む） |
| 合計（実際のメモリースティック使用量） | 約8KB（サムネイル画像PNG含む） |

PSPのセーブデータは最小1ブロック（16KB〜128KB）単位で確保されるため、
実データが小さくても実際の使用容量はブロックサイズに切り上げられる。

## セーブスロット数

- セーブスロット数：**3スロット**
- 各スロットには以下の情報がサムネイル画像として表示される：
  - プレイヤー名・学校名
  - 現在の日程（「修学旅行 X日目」）
  - 制覇数（「XX/47 都道府県制覇」）
  - 累計プレイ時間（「XX時間XX分」）
  - 漢の器レベル（「Lv.XX」）
- 空きスロットは「空きデータ」とアイコン表示

### セーブデータの初期化
- タイトル画面のオプションから「セーブデータ削除」が可能
- 削除前に確認ダイアログを表示（誤操作防止）
- 全スロット削除で初期状態に戻る
