# マスターデータ仕様書 — 喧嘩番長3 全国制覇

## 技マスターテーブル

### テーブル定義

| カラム名 | 型 | 説明 |
|---------|-----|------|
| id | int（主キー） | 技の一意識別子 |
| 技名 | string | 技の表示名 |
| タイプ | enum | 弱攻撃 / 強攻撃 / コンボ / 掴み / ジャンプ / 地元スペシャル |
| 威力倍率 | float | 実ダメージ = 攻撃力 × 威力倍率 - 敵防御力 |
| 消費喧嘩魂 | int | 地元スペシャルのみ使用（通常技は0） |
| 習得条件 | string | 「初期習得」「レベルXX達成」「XX番長を倒す」など |
| ヒット数 | int | 1回の発動で当たる回数（多段ヒット技の場合）|
| 属性 | string | ガード崩し可/掴み無効/無敵フレームあり/壁際追加など |
| コンボ接続 | bool | コンボの途中につなげられるか |
