# GDD — 喧嘩番長4

## ゲームループ概要

本作の基本ゲームループは以下のサイクルで構成される。

```
[フィールド探索]
    ↓
[敵キャラクターとの遭遇] → メンチビーム発射
    ↓
[メンチ対決] → タン語登録 → タンカ入力（リズムゲーム）
    ↓
[先制攻撃 or 被先制] → 戦闘開始
    ↓
[戦闘（通常攻撃/気合技/超気合技）]
    ↓
[勝利 → 校章入手・男気上昇・経験値獲得] or [敗北 → 男気低下]
    ↓
[育成・カスタマイズ] → フィールドに戻る
```

月が進むごとに新エリア・新キャラクターが解放され、ストーリーイベント（ムービー・会話シーン）が進行する。

## メンチシステム詳細仕様

### 概要
フィールド上で敵キャラクターを発見した際に発動するシステム。プレイヤーが敵に向かってメンチビームを放つと、相手もメンチを切り返してくる。このメンチの押し合いの最中にタイミング良くボタン入力を行い「タンカ」を決めることで先制攻撃のアドバンテージを得る。

### 操作仕様
- **発見範囲：** プレイヤー正面約120度、距離10m以内に敵が入るとロックオン可能
- **メンチビーム発射：** Rボタン長押し → 照準ゲージが出現 → 離すと発射
- **メンチ対決ゲージ：** 画面上部に横長のゲージが表示され、双方の「眼力」パラメータで押し合う
- **タン語入手：** マップ上の光るスポット（壁のシミ、看板の落書きなど）にメンチビームを当てると「タン語」カードを入手できる
- **眼力パラメータの影響：** プレイヤーの眼力が相手より高いほどメンチゲージが有利になる

### ゲームへの影響
- メンチ勝利 → タンカフェーズへ移行（先制攻撃チャンス）
- メンチ敗北 → 相手のタンカフェーズへ移行（被先制リスク）
- メンチゲージが互角の場合 → 引き分けで直接戦闘開始

## タンカシステム詳細仕様

### 概要
シリーズ最大のリニューアル要素。従来の「単語組み合わせ選択式」から「リズムに合わせてタイミング良くボタンを押すタイミングゲーム式」に変更された。

### タン語の収集と登録
- フィールドの光るスポットからタン語を入手（単語・フレーズ単位）
- 収集したタン語はタンカ編集画面で△×□○の4ボタンに割り当て登録する
- タンカの長さは1ワード〜8ワードまで設定可能
- 掛け声（開始の叫び）も自由にカスタマイズ可能

### リズム入力仕様
- メンチ勝利後、タンカ選択画面に移行（制限時間あり：約5秒）
- タンカ選択後、画面中央にリズムバーが出現
- 音楽のビートに合わせて流れてくるノーツ（△×□○）を判定ラインでタイミング良く押す
- 判定：PERFECT / GREAT / GOOD / MISS の4段階
- PERFECT以上の連続で「乗り乗りボーナス」が発動し攻撃力が上昇

### タンカの成功・失敗
- 全ノーツをGOOD以上で通過 → タンカ成功：先制攻撃＋ダメージボーナス
- タンカが長いほど成功時ボーナスが高いが、ノーツ数が増えMISSリスクも高まる
- MISSが規定数以上 → タンカ失敗：相手から先制攻撃を受ける
- 「カミカミ」演出：タンカ失敗時は主人公が噛んだ演出が入り笑いを誘う

### 登録タンカ例
| ボタン | タン語例 |
|-------|---------|
| △ | 「やれるもんならやってみろ」 |
| × | 「かかってこいや」 |
| □ | 「オレ様に歯向かうとはいい度胸だ」 |
| ○ | 「この紅南に来たことを後悔させてやる」 |

## 戦闘システム詳細仕様

### 基本概要
1対1・1対多・2対2などの多彩な戦闘形式に対応した3Dベルトスクロール型アクション戦闘。

### 通常攻撃コンボ
- 攻撃ボタン（□）：通常攻撃（3段まで自動コンボ）
- □□□：3段コンボ
- □□△：コンボ→フィニッシュ（吹き飛ばし）
- 掴み（○）：近接掴み → 投げ（方向入力で投げ先変更）
- 背後回り込み（L＋方向）→ □：バックアタック

### 得物システム
- フィールド上のオブジェクト（パイプ・木箱・コーン等）を掴んで武器として使用可能
- 得物使用中は攻撃力上昇だが男気ゲージが徐々に減少
- ガラス・扉などオブジェクトの破壊演出あり
- 得物を使い続けると「シャバゾウ」判定になるリスクあり

### 気合技
- 溜めボタン（R）長押し → 気合ゲージ消費で発動
- 通常攻撃より威力が高く叫び声付きで演出が派手
- 発動中は「息切れ」リスクあり：連発すると一時的に動作が鈍くなる

### 超気合技
- うんこ座りポーズ（↓↓）で気合を溜め始める（数秒のチャージ時間）
- チャージ完了後：専用ボタン入力で発動
- 大ダメージを与える一発逆転技（HPが低い状態でも逆転可能）
- 各キャラクター固有の演出あり（スタイリッシュ系・コミカル系・パワー系）
- レベルアップ要素：同じ超気合技を繰り返し使用すると威力・演出が強化される

### ガード
- Lボタン：ガード構え（連打攻撃を一定数受け止める）
- ガードゲージ超過でガードブレイク → よろめき状態
- 完全無敵ガードはなし（超気合技はガード不能）

## 男気システム詳細仕様

### 男気ゲージ
- 画面右下に常時表示（5段階表示：漢・義・侠・凡・シャバゾウ）
- 最高値「漢」の状態で正々堂々と勝利すると「光る校章」を入手できる

### 男気が上昇する行動
- 1対1の正々堂々とした喧嘩に勝利
- 強い相手（格上）に勝利
- 仲間を助ける行動
- 弱者をかばう行動

### 男気が低下する行動
- 得物を使用した戦闘での勝利
- 一般人への暴力
- 背後からの不意討ち
- 複数対一での喧嘩（多人数で1人を囲む）

### シャバゾウ状態
- 男気ゲージが最下層に達すると「シャバゾウ」扱いとなる
- 周囲の不良から報復攻撃を受けやすくなる
- 一部イベント・NPCの好感度が低下する
- 光る校章が入手できない状態になる

## 成長・パラメータシステム

### 男の器（プレイヤー成長要素）
レベルに相当するシステム。戦闘勝利・イベントクリアで経験値を獲得し、「男の器」が上がる。

| パラメータ | 説明 | 成長方法 |
|-----------|------|---------|
| 腕力 | 与ダメージに直接影響 | 戦闘勝利 |
| 体力 | 最大HPに影響 | 戦闘継続 |
| 眼力 | メンチビームの強さ | メンチ勝利回数 |
| 肺活量 | 気合技の持続・息切れ耐性 | 気合技使用回数 |
| 足の速さ | 移動速度・回避性能 | 逃げた敵の追跡 |
| 根性 | ピンチ時の反撃力上昇 | HP低い状態で勝利 |

## マップ・フィールドシステム

### エリア構成
- 紅南高校内（校舎・体育館・屋上・グラウンド）
- 商店街エリア（餃子の王将・コンビニ・神社・公園）
- 住宅街エリア
- 工場跡地エリア
- 二丁目（チーマーWASP縄張り）
- 郊外道路（暴走族刃華羅縄張り）

### フィールド上の要素
- 光るスポット：タン語入手ポイント
- 神社：喧嘩魂入手・三上広吾の買取場
- たまり場：占領で使用可能になる拠点（装備変更・麻雀等ミニゲーム）
- 仲間呼出：携帯電話で仲間を呼び出せる

## その他システム

### 校章集め
- 紅南高校300名を倒すごとに校章入手
- 学生名簿に記録される（名前・クラス・簡易設定付き）
- 全校章収集で超気合技「漢の拳」入手

### 武勇伝システム
- 特定条件達成でアンロックされる実績・称号
- 例：「最速番長」（1週間以内に10人制圧）「義の男」（男気最高状態で50勝）等

### 休日モード
- メインストーリークリア後に解放
- 全国47都道府県の番長と1対1アドホック対戦可能
- 勝利で相手の超気合技入手・仲間化

### 実在コラボ要素
- 「餃子の王将」：実際のメニューを購入してHPや気合ゲージを回復可能
- アサヒ飲料自動販売機：各種ドリンク購入で一時的なパラメータ強化
