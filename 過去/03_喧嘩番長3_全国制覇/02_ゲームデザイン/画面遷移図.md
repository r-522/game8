# 画面遷移図 — 喧嘩番長3 全国制覇

## Mermaid フローチャート

```mermaid
flowchart TD
    BOOT([起動]) --> TITLE[タイトル画面]
    TITLE --> MAIN_MENU[メインメニュー]
    MAIN_MENU --> NEW_GAME[新規ゲーム開始\nキャラクターカスタマイズ]
    MAIN_MENU --> LOAD_GAME[セーブデータ選択\nロード]
    MAIN_MENU --> NIGHT_OUT[夜遊びモード]
    MAIN_MENU --> AD_HOC[アドホック対戦]
    MAIN_MENU --> OPTIONS[オプション設定]

    NEW_GAME --> OPENING[オープニングイベント\nムービー]
    LOAD_GAME --> FIELD

    OPENING --> FIELD[フィールド探索\n郷都マップ]

    FIELD --> EVENT[イベントシーン\nNPC会話]
    EVENT --> FIELD

    FIELD --> MENCHI[メンチバトル\nゲージ対決]
    MENCHI --> MENCHI_WIN{メンチ\n勝敗}
    MENCHI_WIN -- 勝利 --> TANKA[タンカバトル\n方言セリフ選択]
    MENCHI_WIN -- 敗北/無視 --> FIELD

    TANKA --> TANKA_RESULT{タンカ\n正誤}
    TANKA_RESULT -- 正解 --> COMBAT_PLAYER_FIRST[戦闘開始\n先制攻撃あり]
    TANKA_RESULT -- 不正解 --> COMBAT_ENEMY_FIRST[戦闘開始\n相手先制あり]

    COMBAT_PLAYER_FIRST --> COMBAT[3D戦闘画面]
    COMBAT_ENEMY_FIRST --> COMBAT

    COMBAT --> COMBAT_RESULT{戦闘\n結果}
    COMBAT_RESULT -- 勝利 --> VICTORY[勝利演出\n男気更新]
    COMBAT_RESULT -- 敗北 --> DEFEAT[敗北画面\n体力0]

    VICTORY --> OTOKOGI_UPDATE[男気値更新\nしおり/技入手]
    OTOKOGI_UPDATE --> FIELD

    DEFEAT --> RETRY{リトライ?}
    RETRY -- はい --> FIELD
    RETRY -- いいえ --> MAIN_MENU

    FIELD --> TIME_END{夜19時?}
    TIME_END -- Yes --> INN[旅館\n夜のセーブポイント]
    TIME_END -- No --> FIELD

    INN --> INN_MENU[旅館メニュー]
    INN_MENU --> SAVE[セーブ画面]
    INN_MENU --> CUSTOMIZE[コンボ/装備\nカスタマイズ]
    INN_MENU --> REST[休息\n体力全回復]
    INN_MENU --> STATUS[ステータス確認\n各種パラメータ]
    INN_MENU --> NEXT_DAY[翌日へ進む]

    SAVE --> INN_MENU
    CUSTOMIZE --> INN_MENU
    REST --> INN_MENU
    STATUS --> INN_MENU
    NEXT_DAY --> MORNING_EVENT[午前団体行動\nイベント]
    MORNING_EVENT --> FIELD

    FIELD --> DAY7_END{7日目\n終了?}
    DAY7_END -- Yes --> ENDING[エンディング\n分岐]
    ENDING --> STAFF_ROLL[スタッフロール]
    STAFF_ROLL --> RESULT[クリア結果画面]
    RESULT --> MAIN_MENU

    NIGHT_OUT --> NIGHT_FIELD[夜遊びフィールド]
    NIGHT_FIELD --> MENCHI
    NIGHT_FIELD --> NIGHT_EXIT[夜遊び終了] --> MAIN_MENU

    AD_HOC --> AD_HOC_LOBBY[アドホック\n接続ロビー]
    AD_HOC_LOBBY --> AD_HOC_BATTLE[対戦\n戦闘画面]
    AD_HOC_BATTLE --> AD_HOC_RESULT[対戦結果]
    AD_HOC_RESULT --> AD_HOC_LOBBY
    AD_HOC_LOBBY --> MAIN_MENU
```

## 各画面の説明一覧テーブル

| 画面名 | 概要 | 主な入力操作 | 備考 |
|--------|------|------------|------|
| タイトル画面 | ゲームロゴ・キャラ演出・○ボタンで開始 | ○ボタン | デモ演示あり |
| メインメニュー | 新規ゲーム・ロード・夜遊び・対戦・オプション | 十字キー・○ | 最大3スロット |
| キャラクターカスタマイズ | 名前・学校・出身地・外見設定 | 十字キー・○・△ | 初回のみ |
| オープニングイベント | ムービー＋会話演出で郷都到着を描写 | ○でスキップ | スキップ可 |
| フィールド探索 | 郷都の街を3D視点で移動。ミニマップ表示 | スティック・L/R | メインの探索画面 |
| イベントシーン | NPC会話・物語進行のカットシーン | ○で進める | 分岐選択あり |
| メンチバトル | 視線ゲージを押し込む入力対決 | □連打orタイミング入力 | 制限時間あり |
| タンカバトル | 方言セリフの返答を2〜3択で選ぶ | 十字キー・○ | 正誤で先制決定 |
| 3D戦闘画面 | ビートアップアクション戦闘 | □△○×・L/R | 本作のコアプレイ |
| 勝利演出・男気更新 | 勝利ポーズ・男気変動・アイテム取得表示 | ○で確認 | — |
| 敗北画面 | HP0時の演出、リトライ選択 | ○・× | — |
| 旅館メニュー | セーブ・休息・カスタマイズへの分岐 | 十字キー・○ | 夜のハブ画面 |
| セーブ画面 | スロット選択して保存 | 十字キー・○ | 3スロット |
| ステータス画面 | HP・攻防力・男気・喧嘩魂・制覇数確認 | 十字キー | 詳細確認用 |
| コンボカスタマイズ | 技の順序入れ替え・地元スペシャル設定 | 十字キー・○・△ | — |
| エンディング | 制覇数・ヒロイン親密度で分岐 | ○ | 複数パターン |
| クリア結果画面 | 制覇数・プレイ時間・男気最高値などを表示 | ○ | 周回への誘導 |
| 夜遊びモードフィールド | メインと同一マップ。制限なし戦闘 | — | セーブ不可 |
| アドホック接続ロビー | 近距離PSP間の接続を待機 | — | 1対1対戦のみ |
| アドホック対戦 | 対人戦闘。メインモードと同一戦闘システム | — | ローカル通信 |
| オプション設定 | 音量・操作設定・画面輝度など | 十字キー・○ | — |
