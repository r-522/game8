### 補足説明
- **「夜露死苦」表記：** 各パラメータの数値が最大（300ポイント）に達すると、数字の代わりに「夜露死苦（よろしく）」という特別な表記に変わる粋なUI演出
- **体力・気合の実数値：** パラメータ「体力」や「気合」は300ポイント配分時でも実際のゲーム内HPは約4,649という計算式を通した実数値になる（パラメータ点数≠HP値）
- **全ステータス最大化：** 1周のプレイで獲得できる喧嘩魂（経験値）では全パラメータを夜露死苦にするのには不足する。二周目・やり込みが必要

## 戦闘パラメータ

### 主人公ダメージ計算式（推定）

```
与ダメージ = (攻撃力 × 技倍率) - (敵防御力 × 0.5) + 乱数補正(±5%)
```

| 技タイプ | 技倍率 | 備考 |
|---------|-------|------|
| 弱攻撃（単発） | 0.8 | 4連まで |
| 弱攻撃（4連フィニッシュ） | 1.5 | 4ヒット目 |
| 強攻撃（単発） | 1.2 | 3連まで |
| 強攻撃（3連フィニッシュ） | 2.0 | 3ヒット目 |
| オラオラ（フィニッシュ） | 3.5 | 7連以上後に発動 |
| チャージ攻撃 | 2.5 | ガード崩し付き |
| 掴み・投げ | 1.8 | ガード不可 |
| 気合技（通常） | 4.0 | 気合ゲージ小消費 |
| 超気合技 | 6.0〜10.0 | 技により異なる |

### ガード軽減率
- 通常ガード：被ダメージを60%軽減
- ステップ回避成功時：被ダメージ0（完全回避）
- チャージ攻撃でのガード崩し：ガードを無効化し通常ダメージ

### 気合ゲージ充填量
| 充填方法 | 増加量 |
|---------|-------|
| 敵からダメージを受ける | 受けたダメージの10% |
| 気合溜め（R長押し静止） | 1秒あたり最大値の5% |
| タンカ先制ボーナス時 | 気合ゲージ20%スタート |
| アジト快適度ボーナス | 初期値に+5〜20% |

## 男気パラメータ詳細（シブシャバ度）

### シブシャバ度の概念
「シブ度」と「シャバ度」の二軸で男らしさを評価する独自システム。内部的には単一の数値（-100〜+100）で管理されており、プラス値が「シブ」、マイナス値が「シャバ」に相当すると推定される。

### シブシャバ度の初期値・範囲
| 項目 | 値 |
|------|-----|
| 初期値 | 0（中立） |
| 最大値（シブ神） | +100 |
| 最小値（シャバ神） | -100 |
| シブ神認定閾値 | +80以上（推定） |
| シャバ神認定閾値 | -80以下（推定） |

### 変動値テーブル

**シブ度上昇（プラス方向）**

| 行動 | 変動量（推定） |
|------|-------------|
| 正面メンチ→タンカ→正規勝利 | +3〜+5 |
| タンカバトル全成功 | +2 |
| ボス戦勝利（超気合技なし） | +8〜+10 |
| 弱者（一般人）を助ける | +5 |
| 負けた相手に慈悲をかける | +3 |
| 連続5回シブい行動 | ボーナス+5 |

**シャバ度上昇（マイナス方向）**

| 行動 | 変動量（推定） |
|------|-------------|
| 背後からの不意打ち | -5 |
| タンカ中のYボタン不正行為 | -8 |
| 一般人への脅迫 | -10 |
| 土下座 | -15 |
| 鉄砲玉（ナイフ技）使用 | -12 |
| 強制取り立て使用 | -10 |
| 100万ボルト（スタンバトン）使用 | -8 |

### シブシャバ度による変化

| シブシャバ度レベル | NPCの反応 | 特典・ペナルティ |
|-----------------|---------|--------------|
| シブ神（+80〜+100） | 最大限の尊敬。敵が自ら道を開ける | 特別称号・最強アイテム |
| シブい（+30〜+79） | 敬意を示す。一部敵が戦わず舎弟になる | 一部店舗で割引 |
| 普通（-29〜+29） | 特に変化なし | — |

| シャバ神（-80〜-100） | 敵が笑いながら余裕で向かってくる | 5万円＋カンチョー技 |

## バランス調整方針

### 難易度曲線
ゲーム全体の難易度は「入学〜1年前半：チュートリアル」「1年後半〜2年前半：中難易度の緩やかな上昇」「2年後半〜3年：本格的な高難易度」という三段階のカーブを意識する。

ボスキャラクターのステータスは各章ごとに主人公の平均レベルを想定し設定する。想定レベルよりも高レベルでの挑戦時には難易度が自然と低下し、低レベル挑戦時には歯ごたえのある戦闘になるよう設計。

### 主要ボスの想定難易度設定

| ボス名 | 想定挑戦レベル | HP（推定） | ATK（推定） | 特殊スキル |
|--------|-------------|----------|------------|---------|
| 宇崎 総司 | Lv.3〜5 | 800 | 60 | スピード攻撃（命中率高） |
| 片桐 龍治 | Lv.8〜12 | 1,800 | 120 | 確定防御（一定確率でガード） |
| 神埼 蓮 | Lv.15〜20 | 2,500 | 150 | 反撃スキル（ガード後に反撃） |
| 黒川 道三 | Lv.22〜27 | 3,000 | 160 | HP50%以下で強化形態 |
| 御堂 辰哉 | Lv.28〜35 | 4,500 | 200 | 複数特殊スキル、フェーズ制 |

### レベルデザイン方針
- 主人公の推奨成長速度：各ボス戦直前に「推奨レベル±3」程度に到達するよう喧嘩魂（経験値）を設計
- パラメータ自由配分の弊害防止：全てを一つのパラメータに集中させる極端なビルドでもクリアは可能だが、バランス配分が最も安定するよう数値を設計
- アジトボーナスの上乗せ：最大快適度達成時のボーナスは全体の15〜20%程度に留め、アジトなしのクリアも常に可能にする
