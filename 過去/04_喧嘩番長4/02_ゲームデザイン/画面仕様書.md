# 画面仕様書 — 喧嘩番長4

---

## 1. タイトル画面

### 画面概要
ゲーム起動後、スプラッシュ（開発会社・発売元ロゴ）の表示が終わると遷移するゲームの顔となる画面。テーマソング「拳の中のロックンロール」（氣志團）のイントロが流れ、主人公・速水勇太のシルエットと「喧嘩番長4 一年戦争」のロゴが表示される。

### 表示要素
- ゲームロゴ（中央・大）：「喧嘩番長4 一年戦争」
- サブタイトルロゴ（ロゴ下）：英語表記「BANCHOH4: ONE YEAR WAR」
- 背景アニメーション：紅南高校の校門・校舎が夕暮れシルエットで描かれる
- 「PRESS ANY BUTTON」テキスト（下部・点滅）
- 著作権表記（最下部・小フォント）

### 操作方法
- 任意のボタンを押すとメインメニューに遷移
- 30秒放置でデモムービー（ゲームプレイムービー）が自動再生
- デモムービー中に任意ボタンでメインメニューに戻る

### 入力受付
| 入力 | 処理 |
|-----|------|
| 任意ボタン押下 | メインメニューへ遷移 |
| 無操作30秒 | デモムービー再生開始 |

## 2. メインメニュー画面

### 画面概要
ゲームモードの選択を行うハブ画面。背景にはタイトル画面の縮小版が表示され、半透明のメニューパネルが中央に浮かぶ。

### 表示要素
- メニュー項目リスト（縦配置）：
  - ニューゲーム
  - コンティニュー（セーブデータが存在する場合のみ）
  - オプション
  - アドホック対戦（クリア後解放、それまではグレーアウト）
- カーソル（選択中項目をハイライト）
- 背景：タイトルの流用（暗め）

### 操作方法
- 上下方向キー（十字キー）：カーソル移動
- ○ボタン：決定（選択したモードへ遷移）
- △ボタン：各モードの説明表示（ヘルプ）

### 入力受付
| 入力 | 処理 |
|-----|------|
| ↑↓ | カーソル移動 |
| ○ | 決定・遷移 |
| △ | 選択項目の説明表示 |
| START | （機能なし） |

## 3. フィールド画面

### 画面概要
ゲームの主要な舞台となる3Dフィールド画面。主人公を第三者視点（TPS：サードパーソンシューター視点）で操作し、マップ内を自由に探索する。敵キャラクターとのエンカウントがここから始まる。

### 表示要素
- **3Dワールド：** 建物・道路・NPCなどの3Dモデル配置
- **プレイヤーキャラクター：** カメラ後方視点で表示
- **ミニマップ（右上）：** プレイヤー位置・近隣敵の位置を表示（約2cm四方）
- **男気ゲージ（右下）：** 現在の男気ランク表示（漢/義/侠/凡/シャバゾウ）
- **体力バー（左上）：** プレイヤーHP（緑色バー）
- **現在時刻・日付表示（上中央）：** 月・日付・時間帯（午前/午後）
- **タン語所持数表示：** 画面端にアイコン表示
- **イベントマーカー：** ストーリーイベント発生地点に感嘆符「！」表示

### 操作方法
- アナログスティック（左）：移動
- アナログスティック（右）：カメラ回転
- Rボタン長押し：メンチビーム照準モード
- ×ボタン：ダッシュ
- △ボタン：インタラクト（看板読む・NPC話す・拾う）
- ○ボタン：たまり場入場
- STARTボタン：ポーズメニュー

### 入力受付
| 入力 | 処理 |
|-----|------|
| 左スティック | 移動（前後左右） |
| 右スティック | カメラ操作 |
| R長押し | メンチビーム照準モード移行 |
| ×ボタン | ダッシュ |
| △ボタン | インタラクト |
| ○ボタン | たまり場入場 |
| START | ポーズメニュー表示 |

## 4. メンチ対決画面

### 画面概要
敵とのエンカウント時に発生する前哨戦画面。主人公と敵キャラクターが向き合い、互いにメンチビームを放つ演出の後、眼力ゲージの押し合いが始まる。

### 表示要素
- **主人公（左画面）：** 正面向きのバスト画像・表情アニメーション
- **敵キャラクター（右画面）：** 正面向きのバスト画像・表情アニメーション
- **メンチゲージ（画面中央横長）：** 左（主人公側）・右（敵側）に分かれた押し合いゲージ
- **眼力値比較表示：** 双方の眼力数値を数秒間表示
- **メンチビームエフェクト：** 双方の眼から白熱したビームが交差するアニメーション
- **「勝負！」テキスト：** 対決開始時に画面中央から飛び出す演出

### 操作方法
- ○ボタン連打：メンチゲージを自陣側に押し込む
- 一定時間でゲージ値が確定し結果判定

### 入力受付
| 入力 | 処理 |
|-----|------|
| ○連打 | メンチゲージ増加（自陣側に押す） |
| タイムアップ | 現在のゲージ値で勝敗判定 |

## 5. タンカ（リズムゲーム）画面

### 画面概要
メンチ勝利後に発動するリズムゲーム型のタンカシステム画面。登録済みタン語を選択後、音楽のビートに合わせてボタン入力を行う。

### 表示要素
- **タン語選択パネル（最初の5秒）：** △×□○の4ボタンに割り当てられたタン語を表示
- **リズムバー（中央横）：** ノーツ（△×□○のアイコン）が右から左へ流れてくる
- **判定ライン（左端の赤い縦線）：** ここでノーツを押すと判定される
- **コンボカウンター（上部）：** 連続GOOD以上の回数
- **判定テキスト（中央フラッシュ）：** PERFECT / GREAT / GOOD / MISS
- **セリフ表示エリア（下部）：** タン語の文字が逐次表示される（カラオケ字幕風）
- **主人公の叫び演出（左端）：** タンカ成功中は主人公が叫ぶアニメーション

### 操作方法
- △×□○：流れてくるノーツアイコンと同じボタンを判定ラインのタイミングで押す
- 制限時間：タン語の長さに応じて変動（短いタン語で約3秒、長いタン語で約8秒）

### 入力受付
| 入力 | 処理 |
|-----|------|
| △×□○ | 対応ノーツの判定 |
| タイムオーバー | 未入力ノーツをMISS判定 |

## 6. 戦闘画面

### 画面概要
実際の喧嘩を行う3Dアクション画面。単体戦・集団戦両対応。カメラは自動追尾のサードパーソン視点。

### 表示要素
- **プレイヤーHP バー（左上）：** 緑→黄→赤のグラデーション変化
- **敵HP バー（右上）：** 赤色バー（ボス戦は顔アイコン付き）
- **気合ゲージ（プレイヤーバー下）：** 青色バー。気合技使用で消費
- **敵残り人数（右上隅）：** 集団戦時の残り人数カウント
- **男気ゲージ（右下）：** 戦闘中の行動で変動するリアルタイム表示
- **超気合技チャージ表示（左下）：** うんこ座りチャージ時の進捗ゲージ
- **コンボ数テキスト（中央下）：** 「○○ HIT！」表示

### 操作方法
- 左スティック：移動・方向入力
- □ボタン：通常攻撃
- △ボタン：強攻撃（気合技）
- ○ボタン：掴み・投げ
- Lボタン：ガード
- Rボタン：ターゲット切り替え
- ↓↓入力：うんこ座りチャージ開始（超気合技準備）
- チャージ中に△：超気合技発動

### 入力受付
| 入力 | 処理 |
|-----|------|
| □ | 通常攻撃（3段コンボまで） |
| □□△ | コンボフィニッシュ |
| △（気合ゲージあり） | 気合技発動 |
| ○ | 掴み（近距離時）/ インタラクト |
| L | ガード |
| R | ターゲット変更 |
| ↓↓ | うんこ座りチャージ開始 |
| チャージ後△ | 超気合技発動 |
| START | 一時停止（緊急のみ、戦闘中ポーズ） |

## 7. ステータス画面

### 画面概要
プレイヤーの現在のパラメータ・収集状況・習得技を確認するための管理画面。ポーズメニューまたはたまり場から呼び出せる。

### 表示要素
- **キャラクタービュー（左半分）：** 現在の主人公外見の3Dビュー（回転可能）
- **パラメータ表示（右上）：** 腕力・体力・眼力・肺活量・足の速さ・根性の現在値とバー表示
- **男の器レベル（上部）：** 現在レベルと次レベルまでの経験値
- **校章コレクション（タブ切り替え）：** 入手済み校章数 / 総数（300個）、名簿表示
- **超気合技一覧（タブ切り替え）：** 習得済み超気合技の名称・レベル・入手条件
- **武勇伝（タブ切り替え）：** 解放済み称号・未解放称号（??表示）
- **仲間一覧（タブ切り替え）：** 仲間にした番長の顔・名前・連絡先

### 操作方法
- L/Rボタン：タブ切り替え
- 上下方向キー：リスト内スクロール
- ○ボタン：詳細表示
- ×ボタン：画面を閉じてフィールドに戻る

### 入力受付
| 入力 | 処理 |
|-----|------|
| L/R | タブ切り替え |
| ↑↓ | リストスクロール |
| ○ | 項目詳細表示 |
| × | 画面を閉じる |
| 左スティック | キャラクタービュー回転（左タブ選択時） |
