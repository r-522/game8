# 画面仕様書 — 喧嘩番長

---

## 1. タイトル画面

### 画面概要
ゲーム起動後、各社ロゴ表示を経て表示されるタイトル画面。「喧嘩番長」のタイトルロゴと背景イラストが表示される。

### 表示要素
- ゲームタイトルロゴ（中央上部）
- 背景: 夕暮れの昭和風商店街・電柱が並ぶ路地のイラスト
- 点滅テキスト「PRESS START BUTTON」（中央下部）
- コピーライト表示（右下、小文字）

### 操作方法
| ボタン | 動作 |
|--------|------|
| STARTボタン | メインメニューへ遷移 |
| ○ボタン | 同上 |

### 入力受付
- タイトルBGMループ再生中
- デモムービー自動再生タイマー（約90秒放置でオープニングデモ再生）

---

## 2. メインメニュー画面

### 画面概要
ゲーム開始・継続・設定変更を選択するメインハブ画面。

### 表示要素
- 選択肢リスト（縦並び）:
  - 「ニューゲーム」
  - 「コンティニュー」
  - 「スケ番編」（番長編クリア後のみ表示）
  - 「オプション」
- 選択カーソル（点灯表示）
- 背景: タイトル画面と同系統のアートワーク

### 操作方法
| ボタン | 動作 |
|--------|------|
| 十字キー上下 | 選択肢移動 |
| ○ | 選択決定 |
| △ | 前画面（タイトル）へ戻る |

### 入力受付
- ゲームパッド全ボタン受付
- コンティニュー選択時: メモリーカードアクセスが発生し、セーブデータ選択サブメニューへ

---

## 3. フィールド画面

### 画面概要
本作のメイン画面。3Dオープンエリアを自由移動し、不良との遭遇、イベント発生、施設利用などすべてのゲームプレイが行われる。

### 表示要素（HUD）
- 体力ゲージ（左上、赤バー）
- 気合ゲージ（左上、青バー）
- スタミナゲージ（左上、黄バー）
- 男気アイコン（右上、星またはメーター表示）
- ミニマップ（右下、現在位置・周辺地形）
- 時間帯表示（右上）
- 所持金（右下）
- 目標ナビゲーション矢印（画面端）

### 操作方法
| ボタン/スティック | 動作 |
|-----------------|------|
| 左スティック | 移動 |
| 右スティック | カメラ操作 |
| ○ | アクション（話しかける/拾う） |
| □ | ダッシュ |
| R1 | メンチモード起動 |
| SELECT | マップ表示 |
| START | ポーズメニュー |

### 入力受付
- 常時全操作受付
- メンチ範囲内の不良は黄色アイコンで識別表示
- 一般市民はグレーアイコン、女子生徒はピンクアイコン

## 4. メンチ画面

### 画面概要
フィールドでRボタンを押してメンチモードに入ると表示。プレイヤーの目からメンチビームを相手に向ける視線対決画面。

### 表示要素
- プレイヤーキャラクターと対象のクローズアップ表示
- メンチビームのビジュアルエフェクト（目からの光線）
- 気合ゲージ（消費状況リアルタイム表示）
- メンチ成立プログレスバー（照準維持で上昇）
- 「目力（めぢから）」表示（男気に連動した値）

### 操作方法
| ボタン | 動作 |
|--------|------|
| R1長押し | メンチビーム発射・維持 |
| 左スティック | ロックオン対象切替 |
| ○ | ケンカ成立（相手がメンチを返してきた際） |
| △ | メンチキャンセル・フィールドへ戻る |

### 入力受付
- 気合ゲージが0の場合メンチビームが発射できない
- シャバゾウ状態ではビームが相手に届かず自動的にキャンセルされる

## 5. タンカ画面

### 画面概要
メンチ成立後に遷移する啖呵バトル画面。画面上に散らばったセリフの断片を対応ボタンで入力する。

### 表示要素
- 背景: フィールドのぼかし表示
- プレイヤーと敵のバストアップ画像（左右に対面配置）
- セリフテキストの断片が画面中央に散乱表示（各断片にボタンアイコン付き）
- 制限時間タイマーバー（画面上部、時間経過で縮小）
- 入力済みボタンのフィードバック表示（成功時に光る）

### 操作方法
| ボタン | 動作 |
|--------|------|
| □/△/○/× | 対応するセリフ断片の入力 |
| L1/L2/R1/R2 | 裏正解入力（特定タンカで有効） |

### 入力受付
- 制限時間内に正しい順序でボタン入力
- 時間切れ = 失敗扱い（敵に先制攻撃を許す）

---

## 6. 戦闘画面

### 画面概要
タンカバトル後に突入する3Dアクション戦闘画面。1対1または多対多の組み合わせで展開される。

### 表示要素
- プレイヤー体力ゲージ（左上）
- 敵体力ゲージ（右上またはロックオン対象頭上）
- 気合ゲージ（左下）
- コンボカウンター（画面右）
- 技名フラッシュ表示（技発動時）
- ヒットエフェクト（打撃音・星・火花）
- KO/ダウン演出
- 仲間呼び出しアイコン（男気条件達成時に光る）

### 操作方法
| ボタン | 動作 |
|--------|------|
| □ | 弱攻撃 |
| △ | 強攻撃 |
| □□□△ | コンボ例 |
| L1 | 投げ・つかみ |
| ○ | ジャンプ/回避 |
| × | ガード |
| R2 | 怒声 |
| L2+△ | 気合技（事前装備した必殺技） |
| START | ポーズ（戦闘一時停止） |

### 入力受付
- ガード不能技あり（強攻撃の一部）
- うんこ座り: スティックニュートラル + R2長押し（気合回復、体力微減）

## 7. ステータス画面

### 画面概要
自宅「俺様の部屋」またはポーズメニューから参照できるキャラクターの現在状態確認画面。

### 表示要素
- キャラクター名・番長度表示
- 体力（現在値/最大値）
- スタミナ（現在値/最大値）
- 気合（現在値/最大値）
- 男気メーター（シャバ王〜シブイの段階表示）
- 喧嘩慣れ度（経験値バー）
- 装備技リスト（スロット表示）
- 所持アイテム一覧
- 伝説アルバム達成状況（解放済み項目数）
- 現在の服装・髪型プレビュー

### 操作方法
| ボタン | 動作 |
|--------|------|
| 十字キー左右 | タブ切替（パラメータ/技/アイテム/伝説） |
| ○ | 技装備変更（技タブ時） |
| △ | 前画面へ戻る |

### 入力受付
- 技装備変更のみ操作可能（パラメータタブは参照のみ）
- 戦闘中はアクセス不可（ポーズ中も技変更不可）
