## レンダリングシステム

### ポリゴン数制限
3DSのGPU（PICA200）は同時描画ポリゴン数に制限があるため、以下の制限値を設定する。

| カテゴリ | ポリゴン数上限 |
|---------|-------------|
| 主人公キャラクター | 最大4,000ポリゴン |
| 一般敵キャラクター | 最大2,500ポリゴン |
| ボスキャラクター | 最大5,000ポリゴン |
| 背景オブジェクト（近景） | 最大3,000ポリゴン/オブジェクト |
| 1フレーム合計 | 50,000ポリゴン以内を目標 |

### テクスチャ解像度
| 用途 | 解像度 |
|------|-------|
| キャラクターテクスチャ | 256×256px（拡散マップ） |
| 背景テクスチャ | 512×512px（大型背景）/ 256×256px（小物） |
| UIテクスチャ | 256×256px（アトラス使用） |
| エフェクトテクスチャ | 128×128px |

### 立体視（3D）対応
- 上画面のみ裸眼3D立体視対応
- 視差（parallax）値は+15〜-5の範囲で調整
- 戦闘カットインとタンカバトルのキャラ演出で立体感を最大活用
- 3D強度スライダーに完全対応（0=平面2D〜最大強度の動的切替）

### シェーダー
- セルシェーディング（トゥーンシェーダー）を基本とし昭和漫画的な表現を実現
- アウトライン描画（2パスレンダリング）：エッジを太くして漫画的に
- キャラクターは3DS固定機能シェーダーを最大活用

## 物理演算・衝突判定

### 衝突判定方式
- キャラクター間：カプセル型コリジョン
- 攻撃判定：ボックス型またはスフィア型ヒットボックス（技ごとに設定）
- 地形：高さマップ（HeightMap）ベースの地面判定

### 衝突判定の更新頻度
- 毎フレーム（60fps）更新
- 攻撃ヒットボックスは攻撃モーションのアクティブフレームのみ有効化

### フレーム管理
- 固定60fps動作を目標
- 3D表示オン時に処理負荷が上がる場合は3D非表示時との負荷差を最大20%以内に抑える

## メモリ管理方針（128MB RAM 制約への対応）

### メモリ割り当て計画（推定）
| 用途 | 割り当て量 |
|------|----------|
| ゲームプログラム | 16MB |
| テクスチャデータ（VRAM+RAM） | 24MB |
| 3Dモデルデータ | 12MB |
| オーディオデータ（PCM展開済み） | 16MB |
| サウンドバンク | 8MB |
| ボイスデータ（ストリーミング） | 4MB |
| シナリオ・テキストデータ | 4MB |
| ゲームロジック作業領域 | 8MB |
| OSシステム予約 | 36MB |
| **合計** | **128MB** |

### ストリーミング方針
- BGMはゲームカードからストリーミング再生（ループ対応、シームレスループ）
- ボイスデータは1セリフ単位でオンデマンドロード
- ステージ遷移時にテクスチャ・モデルを非同期ロード（ロード画面表示中に実行）

### オブジェクトプーリング
- 敵キャラクターモデルはプール管理（最大8体の同時描画を上限）
- エフェクトオブジェクトはプール管理（最大32個）
- ダメージエフェクト・テキストもプール化

## AI実装方針

### 敵AIの状態機械（FSM）

```
[巡回] → メンチ射程内に主人公 → [警戒] → 主人公が近づく → [追跡]
[追跡] → 戦闘距離 → [戦闘] → 敵HP0 → [ダウン]
[戦闘] → 主人公が離れすぎ → [巡回]
```

### 戦闘AI行動パターン
- **下っ端敵：** 単純な近接→通常攻撃ループ。ガードなし
- **中堅敵：** 近接→弱コンボ3〜5ヒット。25%確率でガード
- **幹部・ボス：** 行動パターンテーブルを参照。フェーズ制で行動が変化。ガード率60%

### CPU難易度スケーリング
- プレイヤーのレベルに応じてAIの反応速度（攻撃インターバル）を動的調整
- ボス専用AIは固定難易度（スケーリングなし）

## サウンドシステム

### サウンドアーキテクチャ
- **BGM：** ADPCM圧縮ストリーミング再生（ステレオ）
- **SE：** PCM8またはPCM16の波形バンク方式（最大32音同時再生）
- **ボイス：** ADPCM圧縮ストリーミング（1チャンネル独立）

### シームレスBGMシステム
- 戦闘開始時にフィールドBGMから戦闘BGMへクロスフェード（0.5秒）
- 戦闘終了時に戦闘BGMからフィールドBGMへクロスフェード（1.0秒）

### 立体音響
- 3DS内蔵ステレオスピーカーの特性を活かした擬似3Dサウンド
- 敵の接近方向からの効果音にパン設定を適用

## ロード最適化

### ロードポイントと非同期処理
- エリア間移動時：フェードアウト中に次エリアのアセットを非同期プリロード
- ストーリーイベント開始前：次シーンのボイス・テクスチャをバックグラウンドロード

### キャッシュ戦略
- 直近2エリア分のデータをメモリに保持
- 3エリア以上離れたアセットはアンロード

### ロード時間目標
| シーン遷移 | 目標時間 |
|----------|---------|
| タイトル→フィールド | 3秒以内 |
| フィールド→戦闘 | 1秒以内（同エリア内戦闘） |
| エリア間移動 | 5秒以内 |
| ゲーム起動からタイトル | 8秒以内 |
