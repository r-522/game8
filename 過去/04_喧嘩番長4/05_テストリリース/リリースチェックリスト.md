# リリースチェックリスト — 喧嘩番長4

## 機能完了確認

### ゲームプレイ機能
- [ ] 全9章（4月〜3月）のストーリーがクリア可能であることを確認
- [ ] 全22ステージのボスが出現し、撃破後に正常な遷移が行われることを確認
- [ ] 全校生徒300名を順次撃破でき、学生名簿が完成することを確認
- [ ] メンチシステム（ビーム発射・ゲージ対決・タン語入手）が全シーンで正常動作
- [ ] タンカシステム（リズムゲーム・全ノーツパターン）が正常動作
- [ ] 戦闘システム（通常攻撃・気合技・超気合技・ガード・得物）が全て動作
- [ ] 男気ゲージ（上昇・低下・シャバゾウ判定）が仕様通り動作

### エンディング確認
- [ ] 通常エンディング（阿久津撃破後）が正常に再生される

- [ ] エンドロール（テーマ曲フルバージョン）が最後まで再生される
- [ ] クリア後に休日モード（アドホック対戦）が解放される

### 全エンディング・全分岐の確認
- [ ] ヒロイン3名（白鳥・澤村・弘田）の各デートイベントが発生する
- [ ] クリスマスイベントが好感度条件に応じて分岐する
- [ ] 全47都道府県のCPU番長が正常に出現しアドホック戦闘ができる
