# GDD — ゲームデザインドキュメント 東京Bro（喧嘩番長-Crash Battle-）

## ゲームループ概要

本作のゲームループは「スタミナ消費→クエスト選択→バトル→報酬獲得→キャラ育成→次のクエスト」という典型的なスマートフォンRPGサイクルを踏襲しつつ、バトルパートに独自のひっぱりアクションを組み込むことで差別化している。

```
ホーム画面
  ├─ クエスト選択（スタミナ消費）
  │    ├─ 通常クエスト（シングルプレイ）
  │    ├─ 協力バトル（最大4人マルチ）
  │    └─ イベントクエスト（期間限定）
  ↓
バトル画面
  ├─ ひっぱりアクション（通常攻撃）
  ├─ メンチビーム（特殊入力）
  ├─ タンカバトル（特殊入力）
  └─ 超気合技（必殺技発動）
  ↓
バトル結果
  ├─ 報酬（経験値・キャラ・アイテム・校章）
  └─ 男気更新
  ↓
育成・強化
  ├─ キャラ合成・進化
  ├─ チーム編成
  └─ ガチャ（校章消費）
```

## メンチシステム詳細仕様

### 概要
「メンチを切る」とは相手を睨みつけてケンカを売る行為を指すシリーズ伝統のシステム。バトル中に一定条件を満たすと「メンチビーム」が発動可能となる。

### 発動条件
- バトル開始から規定ターン経過後に使用可能ゲージが満タンになる
- 一部キャラの超気合技起動前のトリガーとして機能

### 操作仕様
1. 「メンチ切り！」の表示と共にバラバラのひらがな・漢字が画面に出現
2. バラバラになった言葉の断片を正しい文章順にタップ
3. 正しい順番で完成：メンチビーム発動成功 → 超気合技への移行権獲得
4. タイムアウトまたは誤タップ：失敗 → 通常バトルに戻る

### 入力仕様
- タイムリミット：約5〜8秒（推定）
- 出現断片数：4〜8個（難易度によって変化）
- フレーズ例：「俺に勝てると思ってんのか」「男なら逃げんじゃねえ」「お前が相手だろうが関係ねえ」

## タンカシステム詳細仕様

### 概要
「タンカを切る」とは相手を言葉で脅す・啖呵を切る行為。メンチビーム成功後に続けて発動するシリーズ伝統のシステム。

### 発動条件
- メンチビーム成功後に自動的にタンカバトルモードへ移行
- または規定ターン経過後に独立して使用可能

### 操作仕様
1. 「タンカバトル！」の表示と共に画面にセリフ断片が出現
2. メンチ同様にバラバラの言葉を正しい順にタップ
3. 成功：超気合技（キャラ固有必殺技）が発動
4. 失敗：チャージが無駄になり次の機会を待つ

### タンカ切りとの違い
- タンカバトル：プレイヤーキャラが先にタンカを切る形式
- タンカ切り：敵のタンカに対してより早く入力で返す形式（一部ボス戦で発生）

## 戦闘システム詳細仕様

### 通常攻撃（ひっぱりアクション）

**基本操作：**
1. 画面の任意の場所に指を置く
2. 任意の方向に指を引っ張る（矢印が方向と攻撃力を表示）
3. 指を離すと選択中のキャラクターが勢いよく飛び出す
4. キャラが敵に接触するたびにダメージを与える

**ダメージ計算：**
- ダメージ = 引っ張り距離（比例係数）× キャラ攻撃力 × 接触数連鎖倍率
- 接触1回目：基本ダメージ
- 接触2回目：×1.2倍
- 接触3回目以降：+0.2倍ずつ累積上昇

**キャラ特性：**

| 特性 | 効果 |
|------|------|
| 貫通 | 敵を突き抜けて複数体にダメージ |
| 反射 | 壁や障害物に当たって跳ね返りさらに攻撃 |
| 爆発 | 着弾時周囲の敵に範囲ダメージ |
| 回復 | 味方キャラのHPを回復 |
| 強化 | 味方の次の攻撃ダメージをアップ |

**アイテム出現：**
バトル中にランダムでアイテムが出現し、タップすることで効果を得る。

### 超気合技（必殺技）
メンチビーム＋タンカバトル成功後に発動するキャラ固有の必殺技。
- 連続攻撃タイプ：キャラが画面内を縦横無尽に走り回り敵を次々撃破
- 一撃集中タイプ：ボスに超大ダメージを与える単発強烈攻撃
- 全体攻撃タイプ：全敵に均等ダメージを与えるスキル

### 協力プレイ
最大4人でのリアルタイム協力バトル。各プレイヤーが自分のキャラを操作し、連鎖・連携攻撃でボス撃破を目指す。通信はサーバー経由のリアルタイム同期。

## 男気システム詳細仕様

### 男気ゲージ
プレイヤーの「シブさ（男気値）」を示すゲージ。MAXが「シブい」、MINが「シャバい」。

### シブさ上昇行動
- 正々堂々の真っ向勝負での勝利：+高
- 敵の挑発に堂々と応じる：+中
- 困っている味方を救助する：+中
- 武器なし・正面からの戦闘完勝：+低〜中

### シャバさ転落行動
- 武器を使った攻撃：-中
- 不意打ち・奇襲攻撃：-高
- 一般人への暴力：-高（ゲーム内でのペナルティ大）
- 逃げを選択した回数：-低

### 男気値の影響
| 状態 | ゲームへの影響 |
|------|--------------|
| シブい（上位30%） | NPCの好意的反応、特典キャラ入手チャンス、専用エンディング解放 |
| 普通（中間40%） | 標準的なゲーム進行 |
| シャバい（下位30%） | NPCからの冷遇、一部キャラの協力拒否、ペナルティイベント |

## 成長・パラメータシステム

- **キャラレベル**：バトルでの経験値獲得でLv.アップ。最大Lv.は推定50〜99。
- **合成**：同キャラまたは素材キャラを合成して経験値を大量付与
- **進化**：規定Lv.到達＋進化素材で★数アップ。ステータス大幅上昇と外見変化
- **スキルLv.**：キャラの超気合技を重ねることでスキルLv.アップ、威力上昇

## マップ・フィールドシステム

- クエストは画面タップで選択する「ステージ選択型」
- ワールドマップは紅南高校の校内図を模したUI
- 各エリア（教室棟・体育館・屋上・グラウンド等）がノードとして存在
- ストーリー進行でエリアが解放されていく
- 協力バトル用の特殊エリアは通常ストーリーとは別枠

---

## その他システム

- **ガチャ（校章ガチャ）**：ゲーム内通貨「校章」でレアキャラを入手
- **スタミナ制**：自然回復（時間経過）または校章で即時回復
- **デイリーミッション**：毎日ログインで校章・アイテムを入手
- **フレンド機能**：フレンドキャラをサポートとして借用可能
- **ランキング**：バトルスコアで全国プレイヤーとランキング競争
