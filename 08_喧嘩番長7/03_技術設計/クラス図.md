# クラス図 — 喧嘩番長7「波嵐烈伝」

## 主要型・インターフェース定義

### GamePhase（ゲーム状態遷移）
```
type GamePhase = 
  'title' | 'field' | 'menchi' | 'tanka' | 'battle' | 'result' | 'menu'
```

### PlayerStats
```
interface PlayerStats {
  level: number
  exp: number
  hp: number
  maxHp: number
  sp: number
  maxSp: number
  atk: number
  def: number
  spd: number
  ganriki: number   // 眼力
  otokogi: number   // 男気 0-100
}
```

### EnemyData
```
interface EnemyData {
  id: string
  name: string
  faction: string   // 'arasuna' | 'tekkiku' | 'shiramine' | 'yotaka' | 'arashi'
  type: 'mob' | 'mid' | 'boss'
  position: [number, number, number]
  hp: number
  maxHp: number
  atk: number
  def: number
  spd: number
  isBoss: boolean
  isDefeated: boolean
}
```

### TankaFragment
```
interface TankaFragment {
  id: string
  text: string        // 表示テキスト
  order: number       // 正しい順序
  placed: boolean     // 配置済みか
  position: { x: number; y: number }  // UI上の散布位置
}
```

### TankaResult
```
type TankaResult = 'kanpai' | 'kachi' | 'hikiwake' | 'make' | 'kanpai_make'
```

### HitEffect
```
interface HitEffect {
  id: string
  x: number
  y: number
  damage: number
  type: 'normal' | 'strong' | 'super' | 'guard'
  timestamp: number
}
```

### DachiData
```
interface DachiData {
  id: string
  name: string
  faction: string
  effect: string      // 'atk_up' | 'def_up' | 'tanka_assist' | 'field_patrol'
  effectValue: number
  isActive: boolean
}
```

### SkillData
```
interface SkillData {
  id: string
  name: string
  type: 'weak' | 'strong' | 'grab' | 'super' | 'special' | 'support'
  power: number       // ATK倍率
  spCost: number
  description: string
}
```

### ItemData
```
interface ItemData {
  id: string
  name: string
  type: 'consumable' | 'equipment'
  effect: string
  value: number
  quantity: number
}
```

### CutinData
```
interface CutinData {
  skillName: string
  characterName: string
  active: boolean
}
```

## Zustand GameStore

```
GameStore
├── State
│   ├── phase: GamePhase
│   ├── player: PlayerState
│   │   ├── stats: PlayerStats
│   │   ├── position: [x,y,z]
│   │   ├── rotation: number
│   │   ├── animState: string
│   │   └── skills: SkillData[]
│   ├── enemies: Map<string, EnemyData>
│   ├── nearbyEnemyId: string | null
│   ├── currentEnemy: EnemyData | null
│   ├── tankaFragments: TankaFragment[]
│   ├── tankaResult: TankaResult | null
│   ├── combo: number
│   ├── comboTimer: number
│   ├── buchikireGauge: number
│   ├── buchikireMode: boolean
│   ├── hitEffects: HitEffect[]
│   ├── dachi: DachiData[]
│   ├── items: ItemData[]
│   ├── screenShake: { intensity, duration, startTime }
│   ├── hitStop: { active, duration }
│   ├── currentChapter: number
│   └── cutinData: CutinData | null
│
└── Actions
    ├── setPhase(phase)
    ├── setNearbyEnemy(id | null)
    ├── startMenchi(enemyId)
    ├── startTanka(fragments)
    ├── startBattle(tankaResult)
    ├── dealDamageToEnemy(damage, type)
    ├── dealDamageToPlayer(damage)
    ├── addHitEffect(effect)
    ├── removeHitEffect(id)
    ├── incrementCombo()
    ├── resetCombo()
    ├── fillBuchikire(amount)
    ├── activateBuchikireMode()
    ├── endBattle(won)
    ├── gainExp(amount)
    ├── changeOtokogi(delta)
    ├── triggerScreenShake(intensity, duration)
    ├── triggerHitStop(duration)
    ├── addDachi(dachi)
    └── updatePlayerPosition(pos, rot)
```

## コンポーネント依存関係

```
page.tsx
├── GameScene (R3F, ssr:false)
│   ├── CameraRig
│   │   └── reads: player.position, player.rotation, screenShake
│   ├── DistrictLighting
│   │   └── reads: currentChapter
│   ├── DistrictField | BattleArena (conditional on phase)
│   ├── PlayerController
│   │   ├── reads: phase
│   │   └── writes: player.position, player.rotation, nearbyEnemyId
│   ├── CharacterModel (player)
│   │   └── reads: player.animState, player.stats
│   ├── EnemyEntity[] (field phase)
│   │   ├── reads: enemies, nearbyEnemyId
│   │   └── writes: setNearbyEnemy
│   └── EffectComposer
│       └── reads: currentChapter (bloom intensity)
│
├── TitleScreen (phase === 'title')
│   └── actions: setPhase('field'), Supabase signInAnonymously
│
├── FieldHUD (phase === 'field')
│   └── reads: player.stats, enemies, nearbyEnemyId
│
├── MenchiUI (phase === 'menchi')
│   ├── reads: currentEnemy
│   └── actions: startTanka, setPhase('field')
│
├── TankaUI (phase === 'tanka')
│   ├── reads: tankaFragments
│   └── actions: startBattle(result)
│
├── BattleHUD (phase === 'battle')
│   └── reads: player.stats, currentEnemy, combo, buchikireGauge, hitEffects
│
├── BattleSystem (phase === 'battle')
│   ├── reads: player.stats, currentEnemy, buchikireMode, tankaResult
│   └── actions: dealDamageToEnemy, dealDamageToPlayer, endBattle
│
├── ResultScreen (phase === 'result')
│   ├── reads: player (levelup判定), currentEnemy, dachi
│   └── actions: gainExp, backToField, addDachi
│
├── ChapterTitle
│   └── reads: currentChapter, showChapterTitle
│
└── MenuOverlay (menuOpen === true)
    ├── reads: player.stats, dachi, items
    └── actions: savePlayerProgress (Server Action)
```
