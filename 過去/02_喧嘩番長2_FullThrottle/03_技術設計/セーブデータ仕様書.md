# セーブデータ仕様書 — 喧嘩番長2 Full Throttle

## セーブデータ構造定義

### ヘッダ情報

| フィールド名 | 型 | サイズ | 説明 |
|------------|-----|-------|------|
| magic | BYTE[4] | 4B | マジックナンバー "KNK2"（整合性確認） |
| version | UINT16 | 2B | セーブデータバージョン（互換性チェック用） |
| slot_id | UINT8 | 1B | スロット番号（0〜2） |
| timestamp | UINT32 | 4B | 最終セーブ日時（UNIX時刻形式） |
| checksum | UINT32 | 4B | データ整合性チェックサム（CRC32） |

### プレイヤーデータ

| フィールド名 | 型 | サイズ | 説明 |
|------------|-----|-------|------|
| player_name | CHAR[16] | 16B | 主人公名（デフォルト「武田トモヤ」） |
| kans_level | UINT8 | 1B | 漢の器レベル（1〜50） |
| kans_exp | UINT32 | 4B | 現在の漢の器EXP |
| param_hp | UINT16 | 2B | 体力パラメータ（10〜99） |
| param_atk | UINT16 | 2B | 腕っ節パラメータ（10〜99） |
| param_def | UINT16 | 2B | 打たれ強さパラメータ（10〜99） |
| param_spd | UINT16 | 2B | 足の速さパラメータ（10〜99） |
| param_kiai | UINT16 | 2B | 肺活量パラメータ（10〜99） |
| otokogi_score | UINT8 | 1B | 男気スコア（0〜100） |
| kenjin_points | UINT16 | 2B | 未使用の喧嘩魂ポイント |
| gold | UINT32 | 4B | 所持金（円） |

### 装備・技データ

| フィールド名 | 型 | サイズ | 説明 |
|------------|-----|-------|------|
| move_slot[4] | UINT16[4] | 8B | 装備中のコンボ技（スロット0〜3のtechマスターID） |
| move_finisher | UINT16 | 2B | コンボフィニッシャーのtechマスターID |
| move_super | UINT16[3] | 6B | 装備中の超必殺技（最大3つ） |
| learned_moves | BIT[512] | 64B | 習得済み技のビットマップ（id 0〜511の習得状態） |

### ファッションデータ

| フィールド名 | 型 | サイズ | 説明 |
|------------|-----|-------|------|
| fashion_inner | UINT16 | 2B | インナーアイテムID |
| fashion_outer | UINT16 | 2B | アウターアイテムID |
| fashion_mask | UINT16 | 2B | マスクアイテムID（0=なし） |
| fashion_hair | UINT8 | 1B | 髪型ID |

### 進行データ

| フィールド名 | 型 | サイズ | 説明 |
|------------|-----|-------|------|
| current_chapter | UINT8 | 1B | 現在の章（1〜5） |
| current_route | UINT8 | 1B | 0=未決定 / 1=極東連合編 / 2=東関狂走連合編 |
| play_time | UINT32 | 4B | 累計プレイ時間（秒） |
| last_stage_id | UINT8 | 1B | 最後にいたステージID |
| player_position | FLOAT32[3] | 12B | フィールド上の座標（X,Y,Z） |
| game_time | UINT16 | 2B | ゲーム内時刻（0〜1439分=24時間） |

### アイテム所持データ

| フィールド名 | 型 | サイズ | 説明 |
|------------|-----|-------|------|
| item_inventory | UINT8[64] | 64B | アイテムID→所持数のマップ（64種類分） |

### 推定合計サイズ

| 区分 | サイズ |
|------|-------|
| ヘッダ | 15B |
| プレイヤーデータ | 38B |
| 装備・技データ | 80B |
| ファッション | 7B |
| 進行データ | 22B |
| アイテム | 64B |
| フラグ領域 | 512B |
| **合計** | **約738B（推定1ブロック以内）** |

## 進行フラグ一覧

フラグは512ビット（64バイト）のビットマップで管理。各ビットが個別イベント・収集要素の達成状態を示す。

### ストーリー進行フラグ（FLAG_STORY: ビット0〜127）

| フラグID | フラグ名 | 説明 |
|---------|---------|------|
| 0 | INTRO_DONE | オープニング視聴済み |
| 1 | BOSS_YASUO_BEATEN | ヤスオとの最初の戦闘完了 |
| 2 | ROUTE_DECIDED_KYOKUTO | 極東連合ルート確定 |
| 3 | ROUTE_DECIDED_TOSAI | 東関狂走連合ルート確定 |
| 10 | BOSS_TWINS_DEFEATED | 今井双子撃破 |
| 20 | BOSS_KIKUE_DEFEATED | 菊永洋平撃破 |
| 30 | EVENT_KIMURA_RESOLVED | 木村武士謀略事件解決 |
| 50 | BOSS_RYO_DEFEATED | 如月亮撃破（最終戦） |
| 60 | BOSS_HYUGA_DEFEATED | 日向誠撃破 |
| 70 | ENDING_WATCHED | エンディング視聴済み |

### 収集・解放フラグ（FLAG_COLLECT: ビット128〜383）

| フラグID | フラグ名 | 説明 |
|---------|---------|------|
| 128〜235 | LEGEND_01〜LEGEND_108 | 伝説（108の誓い）達成状態 |
| 236〜300 | TECH_UNLOCKED_xxx | 技解放状態 |
| 301〜350 | FASHION_UNLOCKED_xxx | ファッションアイテム解放 |
| 351〜383 | BIKE_UNLOCKED_xxx | バイク種別解放 |

## 保存タイミング

| タイミング | 種別 | 説明 |
|----------|------|------|
| ポーズメニュー → セーブ選択 | 手動セーブ | 任意のタイミングで実行可能 |
| フィールド上の特定オブジェクト（公衆電話等）との接触 | 手動セーブ | フィールド内セーブポイント |
| ボス撃破後のカットシーン終了時 | 自動セーブ | オートセーブとして強制実行 |
| 章の区切りイベント終了時 | 自動セーブ | 章クリア記録 |

## ストレージ使用量

| 項目 | 内容 |
|------|------|
| 対応メディア | PlayStation 2 メモリカード（8MB容量） |
| セーブスロット数 | 3スロット |
| 1スロット使用量 | 約1KBイコール1ブロック（推定） |
| 合計使用量 | 3ブロック（PS2メモリカード1ブロック＝128KB） |
| 要求空き容量 | 最低3ブロック（実質ほぼ無視できる容量） |
| バックアップ | メモリカード間コピー機能はPS2本体機能を利用 |

---

## セーブスロット数

本作は3スロットのマニュアルセーブを提供する。スロット選択画面では以下の情報をプレビュー表示する。

- プレイヤー名・キャラクター顔グラフィック
- 漢の器レベル・男気段階（漢/硬派/普通/軟弱/シャバ）
- 現在の章・ルート（極東連合/東関狂走連合）
- 累計プレイ時間（時：分表示）
- 最終セーブ日時
