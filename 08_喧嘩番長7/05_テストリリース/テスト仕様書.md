## 既知の問題と回避策

| 問題 | 状態 | 回避策 |
|-----|------|-------|
| React 19 + R3F v8 JSX型競合 | 解決済み | `three-jsx.d.ts`のモジュール拡張 |
| npm install peer dep警告 | 解決済み | `--legacy-peer-deps` |
| Supabase server.ts型エラー | 解決済み | 明示的な型注釈追加 |
| ポインターロックのEsc解除 | 既知 | Escキー押下でロック解除→再クリックで再ロック |
