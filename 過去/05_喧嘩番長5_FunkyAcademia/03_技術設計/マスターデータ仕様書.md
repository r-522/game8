## 技マスターテーブル（waza_master）

### カラム定義

| カラム名 | 型 | サイズ | 説明 |
|---|---|---|---|
| id | INT | 4byte | 技ID（一意） |
| name | VARCHAR | 32byte | 技名（日本語） |
| type | TINYINT | 1byte | 技タイプ（1:通常攻撃 2:コンボ 3:超気合技 4:回復 5:補助） |
| power | SMALLINT | 2byte | 基礎威力（0〜999） |
| sp_cost | TINYINT | 1byte | SP消費量（0〜100） |
| learn_condition | VARCHAR | 64byte | 習得条件説明 |
| combo_count | TINYINT | 1byte | コンボヒット数（1〜10） |
| effect_flag | TINYINT | 1byte | 特殊効果フラグ（0:なし 1:ダウン 2:吹き飛び 3:属性付与） |
| unlock_stage | TINYINT | 1byte | 解放必要ステージNo. |
