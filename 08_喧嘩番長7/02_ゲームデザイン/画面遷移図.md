# 画面遷移図 — 喧嘩番長7「波嵐烈伝」

## 全体遷移

```
[タイトル画面]
    │
    ├── ゲームスタート → [匿名認証] → [チャプタータイトル「序章」]
    │                                        │
    │                                        ▼
    │                               [フィールド画面]
    │                                        │
    │                               ─────────────────
    │                               │    フィールド内操作    │
    │                               ─────────────────
    │                               │
    │                               ├── Rキー（敵6m以内）→ [メンチUI]
    │                               │         │
    │                               │         ├── フェイス選択 → [タンカUI]
    │                               │         │         │
    │                               │         │         └── タンカ結果 → [バトル画面]
    │                               │         │                    │
    │                               │         │                    └── [結果画面]
    │                               │         │                             │
    │                               │         │                             └── フィールドへ戻る
    │                               │         │
    │                               │         └── Escキー → フィールドへ戻る
    │                               │
    │                               └── Mキー → [メニュー画面]
    │                                           │
    │                                           ├── ステータスタブ
    │                                           ├── ダチタブ
    │                                           └── セーブタブ
    │
    ├── ランキング → [ランキング画面]
    │
    └── 設定 → [設定画面]
```

## 詳細遷移

### タイトル画面
```
[タイトル画面]
  ├── カーソル上下（↑↓）でメニュー選択
  ├── Enter/Space で決定
  │
  ├── 「ゲームスタート」
  │      → 匿名ログイン（Supabase Auth）
  │      → セーブスロット選択（新規 or 続き）
  │      → チャプタータイトルアニメーション
  │      → フィールド画面
  │
  ├── 「ランキング」
  │      → ランキング画面（スコア上位20件表示）
  │      → Escで戻る
  │
  └── 「設定」（未実装・今後追加）
```

### フィールド画面
```
[フィールド画面]
  ├── WASD/矢印キー: 移動
  ├── Shift: ダッシュ
  ├── R: メンチ（近くに敵がいる場合）
  ├── M: メニュー画面へ
  ├── ポインターロック: カメラ制御
  │
  └── 章イベントトリガー: 特定座標到達 → 会話シーンイベント
```

### メンチUI
```
[メンチUI]
  ├── 方向キー（上下左右）: フェイス選択プレビュー
  ├── Enter/R: フェイス確定 → タンカUIへ
  ├── Esc: キャンセル → フィールドへ
  │
  └── フォーカスビーム演出（選択フェイスを照準）
```

### タンカUI
```
[タンカUI]
  ├── タイマー表示（8秒 or ボス6秒）
  ├── フラグメントカードを方向キーで選択
  ├── Enter: フラグメント配置
  ├── Backspace: 直前配置取消
  │
  ├── タイマー0 or 全フラグメント配置完了
  │      → 評価計算（完勝/勝利/引き分け/敗北/完敗）
  │      → 結果バナー表示（2秒）
  │      → バトル画面へ
  │
  └── Esc: スキップ（自動評価 = 引き分け扱い）
```

### バトル画面
```
[バトル画面]
  ├── J: 弱攻撃
  ├── K: 強攻撃
  ├── I: 掴み
  ├── U: 超気合技（SP50消費）
  ├── L: ガード
  │
  ├── HP0（敵）→ 勝利 → 結果画面
  ├── HP0（自）→ 敗北 → 結果画面
  │
  └── ブチキレゲージ満タン → ブチキレモード自動発動
```

### 結果画面
```
[結果画面]
  ├── 勝利時:
  │      ├── EXPカウントアップアニメーション
  │      ├── レベルアップ → 成長選択UI
  │      ├── ダチ勧誘プロンプト（ボスでない場合）
  │      │      ├── 承諾 → ダチ登録 → 男気+3
  │      │      └── 断る
  │      └── Enter → フィールドへ
  │
  └── 敗北時:
         ├── 男気-8
         ├── 最寄りのたまり場にHP80で復帰
         └── Enter → フィールドへ
```

### メニュー画面
```
[メニュー画面]（Mキーで開閉）
  ├── [ステータス]タブ
  │      ├── HP/SP/ATK/DEF/SPD/眼力表示
  │      ├── 男気ゲージ＋称号表示
  │      └── 装備アイテム変更
  │
  ├── [ダチ]タブ
  │      ├── ダチ一覧（名前・所属・効果）
  │      └── ダチ発動ON/OFF切替
  │
  └── [セーブ]タブ
         ├── スロット1〜3表示（プレイ時間・章進捗）
         ├── セーブ実行 → Supabaseに保存
         └── ロード実行 → Supabaseから読込
```

## 画面ID定義

| 画面ID | 画面名 | GamePhase値 |
|-------|-------|------------|
| SCR_TITLE | タイトル画面 | title |
| SCR_FIELD | フィールド画面 | field |
| SCR_MENCHI | メンチUI | menchi |
| SCR_TANKA | タンカUI | tanka |
| SCR_BATTLE | バトル画面 | battle |
| SCR_RESULT | 結果画面 | result |
| SCR_MENU | メニューオーバーレイ | （field上のオーバーレイ） |
| SCR_RANKING | ランキング画面 | （タイトルからモーダル） |
