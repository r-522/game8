# 画面遷移図 — 喧嘩番長2 Full Throttle

## Mermaid フローチャート

```mermaid
flowchart TD
    BOOT([起動・ロゴ表示]) --> TITLE[タイトル画面]
    TITLE --> MAINMENU[メインメニュー]
    MAINMENU --> NEWGAME[新規ゲーム開始\n主人公名入力]
    MAINMENU --> LOADGAME[セーブデータ選択\nロード画面]
    MAINMENU --> OPTIONS[オプション設定画面]
    MAINMENU --> EXTRA[エクストラ\n鑑賞モード]

    NEWGAME --> OPENING[オープニングムービー]
    LOADGAME --> FIELD
    OPENING --> FIELD[フィールド探索画面]

    FIELD --> MENCHI[メンチ画面\nR2ボタン発動]
    FIELD --> EVENTSCENE[イベントシーン\nカットシーン]
    FIELD --> PAUSEMENU[ポーズメニュー]
    FIELD --> STATUSSCREEN[ステータス画面]
    FIELD --> MAPSCREEN[マップ画面]
    FIELD --> SHOPSCREEN[ショップ画面\n道場・コンビニ等]
    FIELD --> BIKEMODE[バイクモード\n移動画面]

    MENCHI --> MENCHI_WIN{メンチ勝敗}
    MENCHI_WIN -->|勝利| TANKA[タンカバトル画面]
    MENCHI_WIN -->|敗北| COMBAT_UNFAV[不利状態から戦闘]

    TANKA --> TANKA_RESULT{タンカ結果}
    TANKA_RESULT -->|正解| COMBAT_ADV[有利状態から戦闘\n先制攻撃ボーナス]
    TANKA_RESULT -->|裏正解| COMBAT_SUPERADV[超有利状態から戦闘\n超先制ボーナス]
    TANKA_RESULT -->|不正解/時間切れ| COMBAT_NORMAL[通常状態から戦闘]

    COMBAT_ADV --> COMBAT[戦闘画面\n3Dブロウラー]
    COMBAT_SUPERADV --> COMBAT
    COMBAT_NORMAL --> COMBAT
    COMBAT_UNFAV --> COMBAT

    COMBAT --> COMBAT_END{戦闘結果}
    COMBAT_END -->|勝利| RESULT_WIN[勝利結果画面\n男気・漢の器更新]
    COMBAT_END -->|敗北| RESULT_LOSE[敗北画面\nコンティニュー選択]
    COMBAT_END -->|ボス撃破| BOSS_EVENT[ボス撃破イベント\nカットシーン]

    RESULT_WIN --> LEVELUP{レベルアップ?}
    LEVELUP -->|YES| LEVELUPSCREEN[レベルアップ画面\n喧嘩魂振り分け]
    LEVELUP -->|NO| FIELD
    LEVELUPSCREEN --> FIELD

    RESULT_LOSE --> CONTINUE{コンティニュー?}
    CONTINUE -->|YES| FIELD
    CONTINUE -->|NO| MAINMENU

    BOSS_EVENT --> STORYEVENT[ストーリーイベント\n分岐選択]
    STORYEVENT --> FIELD

    PAUSEMENU --> SAVESCREEN[セーブ画面]
    PAUSEMENU --> STATUSSCREEN
    PAUSEMENU --> FIELD
    SAVESCREEN --> FIELD

    STATUSSCREEN --> TECHSCREEN[技カスタマイズ画面\nコンボ編集]
    STATUSSCREEN --> FASHIONSCREEN[ファッション変更画面]
    STATUSSCREEN --> FIELD

    EVENTSCENE --> BRANCHSELECT{ルート分岐選択}
    BRANCHSELECT -->|極東連合編| FIELD
    BRANCHSELECT -->|東関狂走連合編| FIELD

    BOSS_EVENT -->|最終章クリア| ENDING[エンディング画面\n複数分岐]
    ENDING --> STAFF_ROLL[スタッフロール]
    STAFF_ROLL --> MAINMENU
```

## 各画面の説明一覧テーブル

| 画面ID | 画面名 | 概要 | 主な遷移先 |
|--------|--------|------|-----------|
| TITLE | タイトル画面 | ロゴ・キャッチコピー表示。ボタン入力待ち | メインメニュー |
| MAINMENU | メインメニュー | 新規/ロード/オプション/エクストラ選択 | 各種サブ画面 |
| FIELD | フィールド探索 | 3Dマップを自由移動するメイン画面 | メンチ/イベント/各種画面 |
| MENCHI | メンチ画面 | R2でビーム射出、タイミング判定 | タンカ/戦闘 |
| TANKA | タンカバトル | 穴埋め選択式の口頭挑発ミニゲーム | 戦闘 |
| COMBAT | 戦闘画面 | 3Dアクションブロウラー | 勝利結果/敗北画面 |
| RESULT_WIN | 勝利結果画面 | 経験値・男気・習得技の表示 | フィールド/レベルアップ |
| RESULT_LOSE | 敗北画面 | コンティニュー確認 | フィールド/メインメニュー |
| STATUSSCREEN | ステータス画面 | パラメータ・所持技・男気の確認 | 技カスタマイズ/ファッション |
| TECHSCREEN | 技カスタマイズ | 4発コンボの各ヒットに技を割り当て | ステータス画面 |
| SAVESCREEN | セーブ画面 | セーブスロット選択・書き込み | フィールド |
| LEVELUPSCREEN | レベルアップ | 喧嘩魂ポイントを5パラメータに振り分け | フィールド |
| SHOPSCREEN | ショップ | 道場（技購入）・コンビニ（アイテム）等 | フィールド |
| MAPSCREEN | マップ | 全体マップ・現在位置・目的地確認 | フィールド |
| FASHIONSCREEN | ファッション変更 | 上半身インナー・アウター・マスク変更 | ステータス画面 |
| BIKEMODE | バイク移動 | バイク/自転車乗車中の移動画面 | フィールド |
| EVENTSCENE | イベント/カットシーン | ストーリー進行シーン・選択肢 | フィールド/分岐 |
| ENDING | エンディング | ルートと成り上がり度による複数エンディング | スタッフロール |
| OPTIONS | オプション | BGM/SE音量・画面設定等 | メインメニュー |
| EXTRA | エクストラ | クリア後の鑑賞モード・隠しコンテンツ | メインメニュー |
