# 画面仕様書 — 喧嘩番長2 Full Throttle

---

## 画面1: タイトル画面

### 画面概要
ゲーム起動後、メーカーロゴ（ワイズケイ・スパイク）表示を経てタイトル画面に遷移する。タイトルロゴ・キャッチコピー・ゲームタイトルをPS2の標準解像度（512×448）で表示する。

### 表示要素
- ゲームタイトルロゴ（「喧嘩番長2 Full Throttle」）画面中央上部
- キャッチコピー（「忍び寄る、動乱の足音…」）タイトル下
- 「PRESS START BUTTON」点滅テキスト（画面下部）
- 背景：極東市夜景＋二大勢力が睨み合うシルエット演出
- BGM：タイトルテーマ（「男の勲章」アレンジ版）

### 操作方法
- STARTボタン → メインメニューへ遷移
- 一定時間入力なし → デモムービー自動再生

### 入力受付
| ボタン | 処理 |
|--------|------|
| START | メインメニューへ遷移 |
| ×（無操作タイムアウト） | デモムービー再生開始 |

---

## 画面2: メインメニュー

### 画面概要
ゲームの主要モードへのエントリーポイント。セーブデータの有無に応じてロード項目の活性/非活性を動的に切り替える。

### 表示要素
- メニュー項目リスト（縦並び）
  - 「はじめから」
  - 「つづきから」（セーブデータ無し時はグレーアウト）
  - 「オプション」
  - 「エクストラ」（特定条件解放後に表示）
- 背景：フィールドのアンビエント映像（ループ）
- BGM：メインメニューBGM

### 操作方法
- 十字キー上下: カーソル移動
- ○ボタン: 決定・選択項目へ遷移
- △ボタン: 「つづきから」選択時にセーブスロット一覧へ

### 入力受付
| ボタン | 処理 |
|--------|------|
| 十字キー上下 | カーソル移動 |
| ○ | 選択実行 |
| ×（「はじめから」選択時） | キャンセル（タイトル戻り） |

---

## 画面3: フィールド探索画面

### 画面概要
ゲームのメイン舞台となる3Dオープンワールドフィールド画面。プレイヤーキャラクターを操作して街中を自由に移動・探索する。

### 表示要素（HUD）
- HPゲージ（左上）：赤ゲージ表示
- 気合ゲージ（HPゲージ下）：青ゲージ表示
- 男気ゲージ（画面右端縦型）：「漢」「シャバ」の2段階
- ミニマップ（右下隅）：現在位置・目的地方向矢印表示
- 現在時刻（右上）：時計アイコン＋デジタル数値
- 所持金（右上隅）
- 携帯電話アイコン（仲間呼び出し残回数）
- イベント発生時：感嘆符（!）マークが対象NPCの頭上に表示

### 操作方法
- 左スティック: 移動
- 右スティック: カメラ回転
- ×ボタン: ダッシュ
- R2ボタン: メンチビーム発動
- ○ボタン: 話しかける・調べる
- △ボタン: 携帯電話呼び出し
- L1: ロックオン対象変更
- START: ポーズメニュー
- SELECT: マップ画面

### 入力受付
全ボタン常時受付（イベント発生中は操作無効化）

## 画面4: メンチ画面

### 画面概要
フィールド上でR2を押した際に起動するミニゲーム画面。画面構成がフィールドからメンチ専用のオーバーレイUIに切り替わる。

### 表示要素
- 主人公と対象NPCが向き合うカメラ演出（やや引きのアングル）
- 両キャラクターの目からビームエフェクト
- 中央の「対決ゾーン」：ビームの交差点を示すリング
- 「メンチ！」テキストエフェクト（対決開始時）
- タイミング判定リング（○ボタン入力を要求）
- 対象NPCのオーラ表示（強さの目安）

### 操作方法
- R2長押し: ビーム継続照射
- ○ボタン（判定リングが◎になったとき）: 勝利アクション

### 入力受付
| ボタン | 処理 |
|--------|------|
| R2長押し | ビーム照射維持 |
| R2離し | ビーム中断→フィールドに戻る |
| ○（タイミング正確） | メンチ勝利→タンカへ |
| ○（タイミング外れ） | 失敗判定→ペナルティ |

## 画面5: タンカバトル画面

### 画面概要
メンチ勝利後に発生する穴埋めクイズ形式のミニゲーム画面。画面全体が専用UIに切り替わる。

### 表示要素
- 背景：ヤンキー的なタッチのグラフィック（炎・星・番長文字等）
- お題のタンカ文（画面中央）：「このポチが！俺様の前に___を差し出してみせろ！」など
- 選択肢ボックス（3〜4個）：画面下部に横並びで表示
- 残り時間バー（画面上部）：徐々に減少
- 「TANKA BATTLE！」タイトルテキスト
- キャラクターのアップ演出（発言者顔グラフィック）

### 操作方法
- 十字キー左右/上下: 選択肢にカーソル移動
- ○ボタン: 選択肢確定

### 入力受付
| ボタン | 処理 |
|--------|------|
| 十字キー | 選択肢カーソル移動 |
| ○ | 選択確定（正解/不正解判定） |
| 時間切れ | 自動で不正解扱い |

## 画面6: 戦闘画面

### 画面概要
3Dブロウラー方式の格闘戦闘画面。アクション全般を操作するゲームのコア体験。

### 表示要素
- 主人公HP（左側：赤ゲージ＋数値）
- 敵HP（右側または敵頭上：赤ゲージ）
- 気合ゲージ（左側HP下）：必殺技使用可否の指標
- コンボカウンター（中央下部）：連続ヒット数
- 敵名テキスト（敵HP右）
- ロックオン表示（対象敵にリングエフェクト）
- 仲間共闘時：仲間HPゲージ（左側補助表示）
- 必殺技発動時：カットイン演出（フルスクリーン一時切替）
- 時間経過HUD（バトル経過時間：任意表示）

### 操作方法
- 左スティック: 移動
- □: 通常攻撃（連打コンボ）
- △長押し: チャージ攻撃
- ○: 回避ステップ
- ○+△: 気合必殺技
- L1ダブルタップ: ロックオン
- L2: つかみ・投げ
- R1: ガード
- SELECT: （デバッグ用：本番削除）

### 入力受付
全戦闘入力を常時受付。ヒットストップ（攻撃がヒットした際の一時停止）中は約4〜8フレーム入力バッファリングを行う。

## 画面7: ステータス画面

### 画面概要
主人公のパラメータ・習得技・男気・所持アイテムを確認・管理するメニュー画面。

### 表示要素
- 主人公立ち絵（左側大）：装備中のファッションを反映
- パラメータ一覧（右上）：体力/腕っ節/打たれ強さ/足の速さ/肺活量
- 男気ゲージ（右中央）：シブ/シャバの段階表示
- 漢の器レベル＋経験値バー（右下）
- 所持喧嘩魂ポイント（レベルアップ時振り分け用）
- 習得済み技リスト（技カスタマイズサブ画面）
- 伝説（108の誓い）達成数（右上隅）
- タブ切替：「ステータス」「技」「ファッション」「アイテム」「伝説」

### 操作方法
- L1/R1: タブ切替
- 十字キー: カーソル移動
- ○: 決定
- ×: キャンセル・前画面に戻る

### 入力受付
| ボタン | 処理 |
|--------|------|
| L1/R1 | タブ切替 |
| 十字キー | カーソル/スクロール |
| ○ | 選択実行（技装備・アイテム使用等） |
| × | 前画面へ戻る |
| START | フィールドへ直接戻る |
