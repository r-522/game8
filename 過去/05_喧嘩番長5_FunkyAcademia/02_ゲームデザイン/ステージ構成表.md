## ボスキャラクター詳細

### 高津昇吾（鶴目地区ボス）
- HP：250 / ATK：35 / DEF：15
- 行動パターン：コンボ重視型。3コンボを1セットにして繰り出す。HPが半分以下になると超気合技「ツルメソウル」を発動
- 弱点：回避後の反撃タイミングが一定。読み合いに持ち込むと有利
- 倒した後の台詞：「…お前、結構やるな。見直したぜ」

### 国井晃（外関工業ボス）
- HP：450 / ATK：70 / DEF：30
- 行動パターン：ガード破壊技と大技を組み合わせた実力主義スタイル。赤松・飯塚を倒した後に「俺が全部やってやる」と言って戦闘開始
- 弱点：大技発動前に必ず特定モーションを挟む。見切れると対処しやすい
- 特殊フェーズ：HP30%以下で「弱者を蹴落とすのが理――だろうがっ!」と叫びATK×1.5の激化状態に入る
- 倒した後の台詞：「…俺が間違っていたのか。お前のやり方が、本当の強さなのか」

### 八神真（紅南高校ボス）
- HP：700 / ATK：110 / DEF：50
- 行動パターン：速水に比肩する読み合いスタイル。カウンター技と超気合技「天才の一閃」を織り交ぜる高難易度ボス
- 弱点：超気合技発動前の0.5秒の隙にカウンターを入れるとゲージをリセットできる
- 特殊フェーズ：HP50%以下で台詞「死神と呼ばれた俺の本気を見せてやる」→行動速度1.3倍
- 倒した後の台詞：「…ずっと一人でやってきた。手を差し伸べてくれる奴が来るなんて思ってなかった」

### 速水勇太（小野田外伝ボス）
- HP：800 / ATK：120 / DEF：55
- 行動パターン：前作システムを踏まえた多彩な技構成。アメリカで鍛えた「アメリカン☆ドリーム」を必殺技として持つ
- 特殊演出：前作プレイヤーへのファンサービス台詞「懐かしいな、喧嘩してえ奴がまだいたか」
- 倒した後の台詞：「はははっ！やるじゃねえか。俺の名前を越えていけ」

### 最終ボス
- HP：999 / ATK：140 / DEF：60
- 行動パターン：各勢力の技を組み合わせたコピー戦法。全ての超気合技のコピー版を使用する
- 特殊フェーズ1（HP75%）：鶴目スタイルのコンボ連打
- 特殊フェーズ2（HP50%）：外関スタイルの力押し
- 特殊フェーズ3（HP25%）：紅南スタイルの高速カウンター
- 最終フェーズ（HP10%以下）：全スタイル混合。要全ての対処法
