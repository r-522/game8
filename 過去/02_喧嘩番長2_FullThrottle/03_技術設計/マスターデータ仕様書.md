# マスターデータ仕様書 — 喧嘩番長2 Full Throttle

## 技マスターテーブル

### テーブル定義

| カラム名 | 型 | 説明 |
|---------|-----|------|
| id | INT (PK) | 技識別ID（1〜999） |
| 技名 | VARCHAR(64) | 技の名称 |
| タイプ | ENUM | NORMAL/COMBO/FINISHER/SUPER/THROW/CHARGE |
| 威力 | INT | 基礎ダメージ（腕っ節補正前の値） |
| 消費気合 | INT | 技使用時の気合ゲージ消費量（0=消費なし） |
| スタートアップ（F） | INT | 攻撃判定発生までのフレーム数 |
| アクティブ（F） | INT | 攻撃判定が有効なフレーム数 |
| リカバリー（F） | INT | 攻撃後の隙フレーム数 |
| 習得条件 | VARCHAR(128) | 道場名・漢の器レベル・必要金額等 |
| 備考 | VARCHAR(256) | 特殊効果（吹き飛び・ダウン・のけぞりなど） |

## ステージマスターテーブル

### テーブル定義

| カラム名 | 型 | 説明 |

| id | VARCHAR(8) | ステージID（S01〜S16等） |
