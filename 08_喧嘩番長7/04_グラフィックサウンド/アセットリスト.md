# アセットリスト — 喧嘩番長7「波嵐烈伝」

## 概要
本作はプロシージャル生成（コードによる3Dモデル生成）を基本とし、glTFアセットは使用しない。すべての3Dオブジェクトは`BoxGeometry`・`CylinderGeometry`・`SphereGeometry`・`PlaneGeometry`で構成される。

---

## 3Dキャラクターモデル（CharacterModel.tsx）

### 生成方式
React Three Fiberコンポーネント内でプロシージャルに生成。`THREE.MeshToonMaterial`を使用したセルシェーディング。

| コンポーネント | 構成 | マテリアル |
|------------|------|----------|
| 頭部 | BoxGeometry(0.5,0.5,0.5) | MeshToonMaterial（肌色） |
| 胴体 | BoxGeometry(0.6,0.7,0.3) | MeshToonMaterial（服色） |
| 左腕 | CylinderGeometry(0.1,0.1,0.5) | MeshToonMaterial（服色） |
| 右腕 | CylinderGeometry(0.1,0.1,0.5) | MeshToonMaterial（服色） |
| 左前腕 | CylinderGeometry(0.09,0.09,0.45) | MeshToonMaterial（服色） |
| 右前腕 | CylinderGeometry(0.09,0.09,0.45) | MeshToonMaterial（服色） |
| 左脚 | CylinderGeometry(0.12,0.11,0.6) | MeshToonMaterial（ズボン色） |
| 右脚 | CylinderGeometry(0.12,0.11,0.6) | MeshToonMaterial（ズボン色） |
| 左すね | CylinderGeometry(0.1,0.09,0.55) | MeshToonMaterial（ズボン色） |
| 右すね | CylinderGeometry(0.1,0.09,0.55) | MeshToonMaterial（ズボン色） |
| 頭部アウトライン | ×1.04スケール | MeshBasicMaterial（黒、BackSide） |
| 全パーツアウトライン | ×1.04スケール | MeshBasicMaterial（黒、BackSide） |
| 髪（主人公） | 複数BoxGeometry | MeshToonMaterial（黒） |
| リーゼント（荒砂） | ConeGeometry積み重ね | MeshToonMaterial（茶） |
