# シーケンス図 — 喧嘩番長4

## 1. フィールドでの遭遇〜メンチ開始

```mermaid
sequenceDiagram
    participant P as Player
    participant FS as FieldSystem
    participant E as Enemy
    participant MS as MenchiSystem
    participant UI as UIManager
    participant Audio as AudioManager

    P->>FS: move(direction)
    FS->>FS: updatePosition()
    FS->>E: checkProximity(player)
    E-->>FS: inRange = true
    FS->>E: setAlertState()
    E->>UI: showAlertIcon()
    UI-->>P: 警戒マーカー表示

    P->>FS: pressR（メンチビーム照準）
    FS->>UI: showMenchiAim()
    UI-->>P: 照準カーソル表示

    P->>FS: releaseR（発射）
    FS->>MS: fireMenchiBeam(player, enemy)
    MS->>Audio: playSE("menchi_beam")
    Audio-->>MS: SE再生
    MS->>UI: showMenchiBeamEffect()
    UI-->>P: メンチビームエフェクト表示

    MS->>E: respondMenchi()
    E->>MS: returnMenchiBeam()
    MS->>UI: showMenchiBattleScreen()
    UI-->>P: メンチ対決画面に遷移
    Audio->>Audio: playBGM("menchi_tension")
```

## 2. メンチ勝利〜タンカ開始

```mermaid
sequenceDiagram
    participant P as Player
    participant MS as MenchiSystem
    participant TS as TankaSystem
    participant UI as UIManager
    participant Audio as AudioManager

    loop メンチ対決（制限時間内）
        P->>MS: pressCircle()
        MS->>MS: increasePlayerGauge(+5)
        MS->>UI: updateGaugeDisplay()
    end

    MS->>MS: timeUp() or gaugeMaxed()
    MS->>MS: calculateResult()

    alt プレイヤーゲージ > 敵ゲージ（勝利）
        MS->>UI: showMenchiWinEffect()
        UI-->>P: 「メンチ勝利！」演出
        Audio->>Audio: playSE("menchi_win")
        MS->>TS: startTanka(player, enemy)
        TS->>UI: showTanGoSelectScreen()
        UI-->>P: タン語選択画面表示（5秒）

        P->>TS: selectTanGo(tanGoId)
        TS->>TS: buildNoteSequence(tanGoData)
        TS->>UI: showRhythmBar(notes)
        Audio->>Audio: playBGM("tanka_rhythm")
        UI-->>P: リズムバー・ノーツ流れ始める

    else プレイヤーゲージ < 敵ゲージ（敗北）
        MS->>UI: showMenchiLoseEffect()
        UI-->>P: 「メンチ敗北！」演出
        MS->>TS: startEnemyTanka(enemy, player)
        Note over TS: 敵のタンカフェーズへ（手順は同じ、主体が逆）
    else 引き分け
        MS->>UI: showMenchiDrawEffect()
        MS->>MS: triggerDirectCombat()
        Note over MS: タンカなしで直接戦闘開始
    end
```

## 3. タンカ結果〜戦闘開始

```mermaid
sequenceDiagram
    participant P as Player
    participant TS as TankaSystem
    participant CS as CombatSystem
    participant UI as UIManager
    participant Audio as AudioManager
    participant OS as OtokogiSystem

    loop ノーツ入力
        P->>TS: pressButton(buttonType)
        TS->>TS: judgeNote(buttonType, currentTiming)
        TS->>UI: showJudgement(PERFECT/GREAT/GOOD/MISS)
        UI-->>P: 判定テキスト表示
    end

    TS->>TS: calculateTankaResult()

    alt タンカ成功（MISSが規定数以内）
        TS->>UI: showTankaSuccessEffect()
        Audio->>Audio: playSE("tanka_success")
        TS->>CS: startCombat(player, enemy, advantage: PLAYER_FIRST)
        CS->>CS: applyFirstStrikeBonus(player, bonusDamage)
        CS->>UI: showCombatScreen()
        UI-->>P: 先制攻撃演出→戦闘開始

    else タンカ失敗（噛んだ）
        TS->>UI: showKamiKamiEffect()
        Audio->>Audio: playSE("tanka_fail")
        TS->>CS: startCombat(player, enemy, advantage: ENEMY_FIRST)
        CS->>CS: applyFirstStrikeBonus(enemy, bonusDamage)
        CS->>UI: showCombatScreen()
        UI-->>P: 被先制攻撃演出→戦闘開始
    end

    Audio->>Audio: playBGM("combat_bgm")
    CS->>CS: initCombatLoop()
```

## 4. 戦闘終了〜男気更新〜結果画面

```mermaid
sequenceDiagram
    participant P as Player
    participant CS as CombatSystem
    participant OS as OtokogiSystem
    participant SS as StageManager
    participant UI as UIManager
    participant Audio as AudioManager
    participant Save as SaveSystem

    CS->>CS: checkCombatEnd()

    alt プレイヤーHPが0
        CS->>UI: showDefeatAnimation()
        Audio->>Audio: playSE("defeat_sound")
        CS->>OS: subtractOtokogi(5, BATTLE_LOST)
        OS->>OS: updateRank()
        OS->>UI: updateOtokogiDisplay()
        CS->>UI: showDefeatScreen()
        UI-->>P: 「もう一度挑戦する？」選択肢

    else 全敵HPが0（勝利）
        CS->>UI: showVictoryAnimation()
        Audio->>Audio: playSE("victory_fanfare")
        CS->>CS: calculateBattleBonus()
        Note over CS: 正々堂々判定（得物不使用・不意討ちなし）

        alt 正々堂々判定 = true かつ 男気ランク 漢
            CS->>P: giveShiningChokosho(enemyId)
            UI-->>P: 「光る校章」入手演出
        end

        CS->>P: addExp(expReward)
        CS->>P: addChokosho(enemyId)
        CS->>OS: addOtokogi(battleOtokogiGain, FAIR_WIN)
        OS->>OS: updateRank()

        CS->>SS: markEnemyDefeated(enemyId)
        SS->>SS: checkStageComplete()

        alt ステージクリア条件達成
            SS->>UI: showStageClearEffect()
            SS->>SS: checkChapterComplete()

            alt 章クリア条件達成
                SS->>UI: showChapterClearScreen()
                SS->>SS: advanceChapter()
                SS->>Save: autoSavePrompt()
            end
        end

        CS->>UI: showResultScreen(exp, chokosho, otokogi)
        UI-->>P: 戦闘結果画面表示

        alt レベルアップ条件
            P->>P: levelUp()
            UI-->>P: レベルアップ演出・パラメータ選択
        end
    end
```

## シーケンス処理参加者まとめ

| 参加者 | 役割 |
|-------|------|
| Player (P) | プレイヤーキャラクター。入力を受け取り行動する |
| FieldSystem (FS) | フィールド探索・NPCとの接触判定を管理 |
| Enemy (E) | 敵キャラクター。AI行動・反応を担う |
| MenchiSystem (MS) | メンチビーム発射〜メンチ対決の処理を管理 |
| TankaSystem (TS) | タン語選択・リズムゲーム・判定処理を管理 |
| CombatSystem (CS) | 実際の戦闘処理・ダメージ計算・勝敗判定を担う |
| OtokogiSystem (OS) | 男気ゲージの増減・ランク更新を管理 |
| StageManager (SS) | ステージ進行・章管理・クリア条件判定を担う |
| UIManager (UI) | 全画面の表示更新・演出制御を担う |
| AudioManager (Audio) | BGM・SE・ボイスの再生制御を担う |
| SaveSystem (Save) | セーブデータの読み書きを担う |
