# 画面仕様書 — 東京Bro（喧嘩番長-Crash Battle-）

---

## SCR-001 タイトル画面

### 画面概要
アプリ起動直後に表示される最初の画面。ゲームロゴ、タイトルロゴ、キャッチコピーを表示し、プレイヤーのタップ操作でメインメニューへ誘導する。

### 表示要素
| 要素 | 表示内容 | 位置 |
|------|----------|------|
| 背景 | 紅南高校外観の夕暮れイラスト | 全画面 |
| ゲームロゴ | 「喧嘩番長-Crash Battle-」のタイトルロゴ | 画面中央上部 |
| キャッチコピー | 「つっぱってひっぱって目指せ、最強の漢」 | ロゴ下 |
| タップ促進 | 「タップしてスタート」（点滅アニメ） | 画面中央下部 |
| バージョン情報 | バージョン番号、著作権表示 | 画面最下部 |
| 通信中インジケータ | サーバー接続中はローディング表示 | 右下 |

### 操作方法
- 画面任意の場所をタップ：メインメニューへ遷移（サーバー接続完了後のみ）

### 入力受付
- シングルタップのみ受付
- 多重タップ・マルチタッチは無視

## SCR-002 メインメニュー / ホーム画面

### 画面概要
ゲームのハブ画面。プレイヤーの基本ステータス（スタミナ・校章残高・フレンド数）を常時表示し、各機能へのアクセスを提供する。

### 表示要素
| 要素 | 表示内容 | 位置 |
|------|----------|------|
| プレイヤー情報 | プレイヤー名・ランク・男気ゲージ | 画面上部左 |
| スタミナ | 現在値/最大値・自然回復残り時間 | 画面上部右 |
| 校章残高 | 所持校章数 | スタミナ横 |
| メインビジュアル | リーダーキャラの3Dモデル（アイドルアニメあり） | 画面中央 |
| ナビゲーションボタン | クエスト、ガチャ、キャラ、フレンド、ショップ | 画面下部タブ |
| お知らせバナー | イベント情報・メンテナンス告知 | 画面上部横スクロール |
| デイリーミッション | 本日のミッション達成状況 | 右側ショートカット |

### 操作方法
- 各ナビゲーションボタンをタップで対応画面へ遷移
- メインビジュアルをタップでキャラ詳細へ遷移
- お知らせバナーをタップでお知らせ詳細へ遷移

### 入力受付
- 全ボタンへのシングルタップ
- バナーの横スワイプ

## SCR-003 フィールド / クエスト選択画面

### 画面概要
挑戦するクエストを選択する画面。紅南高校の校内マップをベースにしたUI構成。スタミナ残量と各クエストの消費スタミナを確認しながら選択できる。

### 表示要素
| 要素 | 表示内容 | 位置 |
|------|----------|------|
| マップビジュアル | 紅南高校の俯瞰マップ | 全画面背景 |
| クエストノード | 各エリアの攻略状態（未攻略・攻略済・ボス戦）| マップ上 |
| スタミナ表示 | 現在スタミナ/最大値 | 上部右 |
| クエスト詳細パネル | ノードタップ後に詳細表示（消費スタミナ・敵情報・報酬）| 下部 |
| クリア状態表示 | 3スターまでの評価を星で表示 | 各ノード上 |
| 協力バトルボタン | 協力バトルエリアへの遷移ボタン | 画面右端 |
| イベントバナー | 期間限定イベントへのショートカット | 画面上部 |

### 操作方法
- マップのクエストノードをタップ：クエスト詳細パネルを表示
- 詳細パネルの「挑戦」ボタンをタップ：バトル準備画面へ
- マップをスワイプ：マップスクロール

### 入力受付
- ノードへのシングルタップ
- 詳細パネルのボタンタップ
- マップのドラッグ操作

## SCR-004 メンチ入力画面

### 画面概要
バトル中にメンチビームが発動した際に表示される特殊入力画面。バラバラになった言葉の断片を正しい順番にタップして文章を完成させる。

### 表示要素
| 要素 | 表示内容 | 位置 |
|------|----------|------|
| エフェクト背景 | 目を見開くキャラクターの演出エフェクト | 全画面 |
| 「メンチ切り！」テキスト | 煽り文字で演出 | 画面上部 |
| 言葉の断片パネル | バラバラになった文字・単語（4〜8個） | 画面中央に配置 |
| タイムリミットゲージ | 残り時間を赤→黄→緑で表示 | 上部バー |
| 入力済みスロット | 選択済みの断片を順番に表示 | 下部 |
| キャンセルボタン | 入力中のリセット（スロット内の断片を戻す） | 左下 |

### 操作方法
- 断片パネルをタップ：入力済みスロットに追加（順番に並ぶ）
- 入力済みスロットの断片をタップ：取り消してパネルに戻す
- タイムリミット内に全断片を正しい順で配置してEnter

### 入力受付
- 断片パネルへのシングルタップ（連打防止のため同一パネルへの連続タップは短時間無視）
- マルチタッチ対応（2本指同時タップで複数断片の高速選択可能）

## SCR-005 タンカ入力画面

### 画面概要
メンチビーム成功後に続けて表示されるタンカバトル入力画面。メンチと同様のタップ入力で超気合技発動を目指す。演出はより激しく熱い雰囲気。

### 表示要素
| 要素 | 表示内容 | 位置 |
|------|----------|------|
| 炎エフェクト背景 | 赤・オレンジの炎エフェクトで埋め尽くされた背景 | 全画面 |
| 「タンカバトル！」テキスト | 爆発的なフォントで表示 | 画面上部 |
| セリフ断片パネル | バラバラになったタンカセリフの断片 | 画面中央 |
| タイムリミットゲージ | メンチより短めのタイムリミット | 上部バー |
| 入力スロット | 順番通りに並べるスロット | 下部 |
| 超気合技名表示 | 「成功で○○発動！」という予告表示 | 右上 |

### 操作方法
- メンチ入力画面と同一操作
- 全断片を制限時間内に正しく並べると超気合技演出へ

### 入力受付
- シングルタップ、マルチタッチ対応（メンチと同仕様）

## SCR-006 バトル画面

### 画面概要
ゲームのメインプレイ画面。ひっぱりアクションによる戦闘が行われる。最大4人協力時は各プレイヤーのキャラが同一画面上に表示される。

### 表示要素
| 要素 | 表示内容 | 位置 |
|------|----------|------|
| ステージ背景 | 紅南高校の各エリア3D/2.5D背景 | 全画面 |
| 自キャラ | 操作対象のキャラクター（3Dモデル） | 画面内自由配置 |
| 敵キャラ | 敵キャラクター群（3Dモデル） | 右側エリア |
| HPバー | 自キャラ・敵キャラのHP表示 | 各キャラ上部 |
| 引っ張り矢印 | 指を引っ張った方向・力を示す矢印 | タッチ追従 |
| メンチゲージ | メンチ発動可能ゲージ | 左下 |
| ターン数表示 | 残りターン数カウント | 上部中央 |
| アイテム | バトル中に出現するアイテムアイコン | ランダム配置 |
| 協力プレイヤー情報 | 他プレイヤーのキャラ・HP（協力時） | 左側 |

### 操作方法
- 画面任意の場所を押し続ける：引っ張りモード開始（矢印表示）
- 引っ張りながら方向・距離を調整：攻撃方向・威力を決定
- 指を離す：キャラ発射・攻撃実行
- アイテムをタップ：即時取得

### 入力受付
- 長押し→ドラッグ→リリースの一連の操作
- アイテムへのシングルタップ（他の操作中でも受付）
- ピンチ操作：ズームイン・アウト（対応機種のみ）

## SCR-007 ステータス画面

### 画面概要
プレイヤーおよび所持キャラクターの詳細なステータスを確認する画面。合成・進化へのショートカットも提供する。

### 表示要素
| 要素 | 表示内容 | 位置 |
|------|----------|------|
| キャラモデル | 3Dキャラクターモデル（タップで回転） | 左半分 |
| キャラ名・星数 | キャラクター名と現在の★ | 右上部 |
| レベル | 現在Lv. / 最大Lv.（プログレスバー） | 右上部 |
| ステータス一覧 | HP・攻撃力・特性・超気合技名 | 右中部 |
| 経験値バー | 次のレベルまでの経験値 | 右下部 |
| スキル詳細 | 超気合技の効果・発動条件説明 | 下部 |
| 合成ボタン | 合成画面へのショートカット | 右下 |
| 進化ボタン | 進化条件を満たしている場合に点灯 | 右下 |

### 操作方法
- キャラモデルをドラッグ：360度回転表示
- 「合成」ボタンタップ：合成画面へ遷移
- 「進化」ボタンタップ（条件満了時）：進化確認ダイアログ
- 「戻る」ボタン：キャラ管理画面へ戻る

### 入力受付
- ドラッグ操作（モデル回転用）
- シングルタップ（各ボタン）
