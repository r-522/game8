# GDD — 喧嘩番長6 Soul & Blood

## ゲームループ概要

本作のゲームループは「日常→遭遇→喧嘩→成長→日常」の繰り返しを基本とする。

```
[フィールド探索]
  ↓
[メンチビーム発射]
  ↓
[タンカバトル（セリフ入力）]
  ↓
[3D格闘戦]
  ↓
[勝敗判定・シブシャバ度更新]
  ↓
[経験値・校章・アイテム入手]
  ↓
[ステータス成長・技習得]
  ↓
[アジト帰還・セーブ]
  ↓（ループ）
[フィールド探索]
```

1学期が終わると自動的に次学期へ進行しストーリーが前進する。各学期内では自由行動期間とイベント強制進行期間が交互に訪れる。

## メンチシステム詳細仕様

### 概要
フィールド移動中、Rボタンを押し続けることでメンチビームを発射するモードに入る。主人公の目からビームが放射され、照準を合わせて敵キャラクターの顔に当てることで喧嘩を挑む。

### 動作仕様
- **発動ボタン：** Rボタン（長押し）
- **照準移動：** スライドパッドまたは十字キー
- **ロックオン：** 照準が敵の顔に一定時間留まると自動ロック
- **発射判定：** ロック中にRボタンを離す
- **射程：** 約10〜15キャラクター身長分（推定）
- **命中箇所：**
  - 顔→「ガンの付け合い状態」（同ランクの敵に有効）→タンカへ移行
  - 胴体→軽い威嚇リアクション、タンカ移行せず
  - 背後→不意打ち判定（シブシャバ度低下）

### メンチビームの特性
- 相手が「同じ獣の目を持つ者」（不良・番長クラス）の場合、相手からもビームが放たれ「ガンの付け合い（メンチバトル）」に発展
- メンチビーム中に主人公は移動速度が低下し視野が限定される
- 複数の敵が近くにいる場合、最も危険度が高い（ランクが上の）敵にロックが引き寄せられる傾向
- 舎弟化済みの敵にはメンチビームの代わりにコマンドメニュー（命令系統）が表示される

### シブシャバ判定
- 正面からメンチを当ててタンカへ→シブ度維持
- 背後から接触して不意打ち→シブ度低下
- メンチビームをオフにして接近し奇襲→シブ度大幅低下

## タンカシステム詳細仕様

### 概要
メンチビームによる「ガンの付け合い」成立後、タンカバトルに移行する。スライドパッドの方向入力で威勢の良い口上（タンカ）を繰り出す言葉の戦い。

### 動作仕様
- **入力方式：** スライドパッドの8方向入力
- **表示：** 上画面にキャラクター、下画面に入力ガイドとセリフ
- **制限時間：** 各入力ごとに約1〜2秒のタイムリミット
- **連続成功ボーナス：** 全タンカを成功させると戦闘開始時に先制攻撃権（最初の一撃優位）
- **タン語選択：** タンカバトル最後に4種類のタン語から1つを選択する

### タン語の種類（推定）
1. 「てめえ！」（標準的な挑発）
2. 「うりゃ！」（突撃宣言）
3. 「このやろう！」（怒りの発露）
4. 「かかってこい！」（堂々たる挑戦状）

### シャミセンシステム
- **発動：** タンカバトル中、L+Rボタン同時押し
- **効果：** 複数のセリフを組み合わせた特殊口上で、シブ度を維持したままケンカを回避可能
- **成功条件：** 相手の性格・ランクに合ったセリフ組み合わせを選択する必要あり
- **失敗時：** 通常のタンカバトルに戻る

### セリフカスタマイズ
- タンカバトルで使用するセリフをプレイヤーが一定範囲でカスタマイズ可能
- セリフショップ（ゲーム内通貨で購入）で新しいタンカセリフを入手できる

### シブシャバ判定
- 入力を正確に行い全タンカ成功→シブ度上昇
- Yボタンによる不正行為（ズルをする）→シブ度大幅低下、ただし成功しやすくなる

## 戦闘システム詳細仕様

### 基本操作
| ボタン | 動作 |
|--------|------|
| Y | 弱攻撃（最大4連続入力可） |
| X | 強攻撃（最大3連続入力可） |
| A | 掴み・投げ・ダウン乗り |
| L | ガード |
| X+A | 気合技発動 |
| スライドパッド | 移動 |
| L+スライドパッド | ステップ回避 |

### コンボシステム
- 弱攻撃（Y）と強攻撃（X）を合計7回以上繋げると「オラオラ」が発動可能状態になる
- オラオラ：連続乱打のフィニッシュ技。タイミングよくボタンを押すと炸裂
- チャージ攻撃（長押し強攻撃）：相手のガードを崩せる重要システム
- ジャンプ攻撃・空中コンボは本作では削除（前作から変更）
- コンボ技の組み合わせをプレイヤーが自由にカスタマイズ可能

### 気合技・超気合技
- **気合技：** 気合ゲージを一定量消費して発動（X+A）
- **超気合技：** 気合ゲージを大量消費して発動、ボス撃破で順次習得
- **気合ゲージ充填：** 敵からダメージを受ける、または「気合溜め」アクション（R長押し静止）

| 超気合技名 | 効果 | シブ度影響 |
|-----------|------|-----------|
| 竜巻アッパー | 上昇気流を伴うアッパー | なし |
| 爆裂アイアンクロー | 頭部を掴み振り回し地面叩きつけ | なし |
| 雷雲 | 頭上に多数の落雷を発生 | なし |
| 猛烈タックル | 突進系高威力タックル | なし |
| 鉄砲玉 | バタフライナイフで突刺し | 低下 |
| 連続頭突き | 連続頭突き乱打 | なし |
| 100万ボルト | スタンバトン使用 | 低下 |

### ガード・回避
- ガード（L）：前方からの攻撃を軽減。チャージ攻撃には無効
- ステップ回避：ガード中にスライドパッドで方向入力するとステップ回避
- フリーターゲット：ロックオンなしで直感的に複数の敵を順次ターゲット
- ボス固有スキル：一部ボスは確定防御などの特殊スキルを持ち、超気合技の使い時を見極める必要あり

## 男気システム（シブシャバ度）詳細仕様

### 概念
シリーズ従来の「男気」をリネームした「シブシャバ度」。プレイヤーの行動が「シブい（男らしい）」か「シャバい（卑怯・情けない）」かを2軸で評価する。

### シブ度上昇行動
- メンチビームで正面から挑む
- タンカバトルを正直に行う（ズル入力なし）
- 正々堂々の1対1の喧嘩
- 負けた相手に止めを刺さない慈悲
- 弱者を助ける（一般人を助けるイベント）

### シャバ度上昇行動
- 背後からの不意打ち
- タンカバトル中のYボタン不正行為
- 一般人への脅迫・恐喝
- 土下座
- 特定の気合技使用（鉄砲玉、強制取り立て、100万ボルト）

### シブシャバ度の影響
- **NPCの反応変化：** シブい男には尊敬、シャバい男には侮蔑の態度
- **ストーリー展開変化：** 一部イベントの選択肢が変わる
- **称号・アイテム入手：** シブ神（シブ度最大）、シャバ神（シャバ度最大）で特典入手
- **シャバ神特典：** 5万円（ゲーム内通貨）＋羽交い締め技「カンチョー」入手
- **シブ神特典：** 特別称号と最強クラスのアイテム（詳細未公開）

## 成長・パラメータシステム

### 漢の器（レベルシステム）
- 喧嘩魂（経験値）を蓄積することで「漢の器」が上昇（レベルアップ相当）
- レベルアップ時に4ポイントを獲得、5つのパラメータに自由配分
- 各パラメータの上限は300（表記：「夜露死苦」）
- 体力・気合の実数値は最大4649付近まで上昇
- 全ステータスを最大にするには1周では経験値が足りず、やり込みが必要

### 5つのパラメータ
| パラメータ | 英名（推定） | 主な効果 |
|-----------|------------|---------|
| 体力 | HP | 最大HP増加 |
| 気合 | SP | 最大気合ゲージ増加 |
| 攻撃 | ATK | 物理攻撃ダメージ増加 |
| 防御 | DEF | 被ダメージ軽減 |
| 漢気 | SPIRIT | シブ度の最大値・特殊効果 |

## マップ・フィールドシステム

### フィールド構成
- 華ヶ崎市内を複数エリアに分割した箱庭型フィールド
- 主要エリア：学校（普通科棟・工業科棟・グラウンド）、商店街、住宅街、海岸、郷都（修学旅行先）
- 各エリアを自由に探索可能（ストーリー進行に応じて解放）

### 治安システム
- 各エリアの治安をパーセンテージで表示
- 治安低下：店舗利用不可・敵出現増加・イベント制限
- 治安回復方法：不良・チンピラ・ヤクザを倒す、店舗での買い物
- 全エリア治安100%達成で「光の剣」（ビームサーベル）入手

### アジトシステム
- 学校の空き部室を自分のアジトにカスタマイズ可能
- ソファ・テーブル・照明など快適グッズを購入・設置
- 快適度上昇で主人公のパラメータにボーナス付与

## その他システム

### 校章収集
- 敵（特に九鬼島高校生徒）をメンチビームで倒すと校章を入手
- 全500枚が存在
- 350枚達成で卒業式直前の特別戦闘イベント解放
- 全国ランキングバトルでオンライン対戦に使用

### 舎弟システム
- 校章収集と連動して敵を舎弟化できる
- 舎弟化した敵は以後フィールドで支援してくれる

### バイト・日常活動
- 各学期中の自由行動期間にバイトや日常活動が可能
- 七瀬千尋の海の家バイト手伝いが恋愛好感度に影響

### セリフショップ・アイテムショップ
- ゲーム内通貨（円）でタンカセリフ、装備、消費アイテムを購入
- 治安回復にも買い物が有効
