# 通信プロトコル仕様書 — 喧嘩番長4

## 本作の通信機能について

本作の**メインストーリーモードはシングルプレイ専用**であり、インターネットを通じたオンライン通信機能は実装されていない。通信機能はゲームクリア後に解放される「休日モード（全国番長アドホック対戦）」のみに限定される。

---

## アドホック対戦機能

### 概要

PlayStation Portableに搭載されたIEEE 802.11b（Wi-Fi）規格の**アドホックモード**（PSP同士が直接無線接続する方式）を使用した1対1のリアルタイム対戦。通信はアクセスポイントを経由せず、PSP同士が直接ピアツーピアで通信する。

本機能はメインストーリークリア後（FLAG_0180「HOLIDAY_MODE_UNLOCKED」が立った状態）にのみ解放される。

### 対応機種と通信規格

| 項目 | 仕様 |
|-----|------|
| 対応通信規格 | IEEE 802.11b（2.4GHz帯） |
| 通信モード | アドホックモード（Ad Hoc Mode） |
| 通信距離目安 | 10〜15m（障害物なしの場合） |
| プレイ人数 | 2人（1対1） |
| 対応PSP機種 | PSP-1000 / PSP-2000 / PSP-3000 |
| WLANスイッチ | 起動前にONにする必要あり |

### 接続手順

```
[ホスト側操作手順]
1. 休日モードを選択
2. 「対戦をひらく」を選択
3. PSPのWLANスイッチをONにする
4. 相手プレイヤーを待機（画面に「対戦相手を待っています...」表示）
5. 相手が接続してきたら「○」ボタンで接続許可

[ゲスト側操作手順]
1. 休日モードを選択
2. 「対戦に参加する」を選択
3. PSPのWLANスイッチをONにする
4. 近くのホストを検索（「対戦を探しています...」表示）
5. 見つかったホストリストから選択して接続リクエスト送信
```

### 対戦モード

| モード名 | 内容 |
|---------|------|
| 全国番長決定戦 | 1対1リアルタイム戦闘対戦。勝利で相手から超気合技を入手・仲間化できる |
| 全国47都道府県制覇モード | 全国各都道府県の固有番長（CPU）と対戦してコンプリートを目指す（シングルプレイ内包） |

**注：** アドホック対戦での相手は「実在の対戦相手（他のプレイヤー）」または「休日モード専用のCPU番長キャラクター」の両方に対応。CPU番長は47都道府県それぞれに1名（合計47名）用意されており、シングルプレイでも制覇が可能。

### データ交換仕様

| 交換データ | 方向 | 説明 |
|----------|------|------|
| キャラクタースペック | 双方向 | 対戦相手のパラメータ（腕力・体力・眼力等）をゲーム開始前に交換 |
| カスタム外見データ | 双方向 | 相手キャラクターの外見（髪型・服装等）を表示するための外見データ |
| 入力コマンド | 双方向 | 対戦中の操作コマンドをフレームごとにリアルタイム送信 |
| 対戦結果データ | 双方向 | 勝敗・獲得技情報 |
| 対戦後のプロフィール | 双方向 | 仲間化時のキャラクタープロフィール |

### 通信プロトコル詳細

#### パケット構造

```
[コマンドバイト (1B)] [シーケンス番号 (2B)] [データ長 (2B)] [データ (可変)] [チェックサム (1B)]
```

#### 主要コマンド一覧

| コマンドID | コマンド名 | 方向 | 説明 |
|-----------|-----------|------|------|
| 0x01 | CONNECT_REQUEST | C→H | 接続リクエスト |
| 0x02 | CONNECT_ACCEPT | H→C | 接続許可 |
| 0x03 | CONNECT_REJECT | H→C | 接続拒否 |
| 0x10 | PLAYER_DATA | 双方向 | 対戦開始時のプレイヤーデータ同期 |
| 0x11 | APPEARANCE_DATA | 双方向 | 外見データ同期 |
| 0x20 | INPUT_STATE | 双方向 | フレームごとの入力状態（毎フレーム送信） |
| 0x21 | RANDOM_SEED | H→C | 乱数シード共有（デターミニスティック同期用） |
| 0x30 | COMBAT_EVENT | 双方向 | 攻撃ヒット等の重要イベント通知 |
| 0x40 | BATTLE_RESULT | H→C | 対戦結果送信 |
| 0x50 | PING | 双方向 | 接続確認・遅延計測 |
| 0xFE | DISCONNECT | 双方向 | 切断通知 |

#### 同期方式
- **デターミニスティック同期**（ロックステップ方式）を採用
- 開始前に乱数シードを共有し、双方で同一の演算を実行
- 入力コマンドのみをやり取りし、双方のPSPが独立してシミュレーションを計算
- 1フレーム（1/30秒）を1ステップとして同期
- 遅延が発生した場合：最大3フレームまでの入力バッファを許容

#### エラーハンドリング
- 接続が途切れた場合：3秒以内に再接続を試み、失敗したらLAN対戦結果なしで終了
- パケットロスト：1フレーム前の入力を繰り返し使用（フレームホールド）
- 大幅な遅延（50ms以上）：対戦相手に「通信状態が悪化しています」警告表示

## オンラインランキング

本作にはオンラインランキング機能は**実装されていない**。PSPのインフラストラクチャモード（インターネット通信）は使用しない。

理由：
1. 2010年時点でのPSPオンラインインフラはゲームごとに独自サーバーが必要であり、運用コストが高かった
2. 本シリーズのターゲット層（学生・ローカルプレイ重視）にはアドホック対戦が適切と判断
3. チート・改造対策が困難なため、公正なランキング維持が難しい

## 将来的な通信機能拡張案（参考）

本作では実装しないが、シリーズ次世代作（PS3/PS Vita等）での拡張として以下を検討する。

| 案 | 概要 |
|---|------|
| オンラインランキング | 全国プレイヤーの最速クリアタイム・校章収集数・男気ランクをランキング表示 |
| すれちがい通信 | PSPを持ち歩くと自動で他プレイヤーのキャラクターデータを交換。「出会った番長」コレクション機能 |
| コミュニティ対戦 | インターネット経由で全国のプレイヤーと対戦できるインフラストラクチャ対戦モード |
| ダウンロードコンテンツ | 追加ステージ・追加コスチューム・追加超気合技のDLC販売 |
