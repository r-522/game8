# 画面仕様書 — 喧嘩番長5 漢の法則

---

## 1. タイトル画面（SCR_TITLE）

### 画面概要
ゲーム起動後、PSPロゴ・スパイクロゴのムービー再生後に表示されるゲームタイトル画面。

### 表示要素
| 要素 | 内容 | 位置 |
|---|---|---|
| ゲームロゴ | 「喧嘩番長5 漢の法則」ロゴ画像 | 画面中央上部 |
| キャッチコピー | 「拳で紡げ 漢たちの絆」テキスト | ロゴ下 |
| プレスボタン表示 | 「○ボタンを押してください」点滅テキスト | 画面下部中央 |
| 背景 | 阿弥浜沿線の夜景イラスト、キャラクターシルエット | 全画面 |
| BGM | タイトルBGM（ロック系）ループ再生 | — |
| バージョン表示 | ROM バージョン番号 | 画面右下隅 |

### 操作方法
| 入力 | 処理 |
|---|---|
| ○ボタン / スタートボタン | メインメニュー画面へ遷移 |
| ○ボタン長押し（3秒） | オートデモムービー再生開始 |
| SELECTボタン | なし（無効） |

### 入力受付
- ムービー再生中はボタン入力を受け付けない（ムービースキップ機能なし）
- ムービー終了後から入力受付開始

---

## 2. メインメニュー画面（SCR_MENU）

### 画面概要
ゲームの中枢となるメニュー画面。ゲームのスタートや各種設定変更の起点となる。

### 表示要素
| 要素 | 内容 | 位置 |
|---|---|---|
| メニュー選択肢 | ニューゲーム / コンティニュー / オプション / アドホック通信 | 画面中央 |
| カーソル | 選択中項目をハイライト表示 | 選択項目の左側 |
| 背景 | タイトル画面と共通のイラスト背景（薄暗く処理） | 全画面 |
| BGM | タイトルBGMの継続再生 | — |
| コンティニュー可否 | セーブデータが存在しない場合グレーアウト | コンティニュー項目 |

### 操作方法
| 入力 | 処理 |
|---|---|
| 十字キー上下 | メニュー項目のカーソル移動 |
| ○ボタン | 選択項目を決定・遷移 |
| ×ボタン | タイトル画面に戻る |

### 入力受付
- 画面フェードイン完了後から入力受付開始
- コンティニューはセーブデータ存在時のみ受付

---

## 3. フィールド画面（SCR_FIELD）

### 画面概要
ゲーム本編のメイン画面。阿弥浜沿線の各地区を歩き回り、キャラクターと交流・戦闘する。3Dポリゴン表示（背面視点）。

### 表示要素
| 要素 | 内容 | 位置 |
|---|---|---|
| プレイヤーキャラクター | 3Dモデル、後ろ向き視点 | 画面中央 |
| HPバー | 主人公の現在HP / 最大HP | 画面左上 |
| 気合ゲージ | 超気合技用のゲージ | HPバー下 |
| 男気ゲージ | 顔アイコン付きゲージ（青〜赤グラデーション） | 画面左下 |
| ミニマップ | 現在地・縄張りを示す小型マップ | 画面右下 |
| 地区名表示 | 現在いる地区の名称 | 画面右上 |
| 時間表示 | ゲーム内時間（朝/昼/夕/夜） | 地区名下 |
| 会話テキスト | NPCとの会話テキストボックス | 画面下部 |
| メンチ可能インジケータ | 近くに敵がいる場合点滅アイコン | 画面右中央 |

### 操作方法
| 入力 | 処理 |
|---|---|
| 左スティック | キャラクター移動（8方向） |
| ○ボタン | 決定・会話・施設入場 |
| ×ボタン | キャンセル・走る（押しながら移動で走行） |
| □ボタン | アクション（拾う・押す） |
| △ボタン | 周囲の情報表示 |
| Rボタン | メンチビーム発射 |
| L+○ボタン | 電話画面（ダチ呼出） |
| STARTボタン | ステータス / ポーズメニュー |
| SELECTボタン | マップ画面表示 |

### 入力受付
- イベント・会話中はスティック移動以外無効
- ショップ等の施設に隣接時のみ施設入場の○入力を受付

## 4. メンチ画面（SCR_MENCHI）

### 画面概要
敵との視線対決を行う画面。メンチビームの照射・顔の選択・相手の反応を確認する演出的な対決画面。

### 表示要素
| 要素 | 内容 | 位置 |
|---|---|---|
| 主人公のアップ | 主人公の顔・表情のクローズアップ | 画面左半分 |
| 相手のアップ | 敵キャラクターの顔クローズアップ | 画面右半分 |
| メンチゲージ | 両者のメンチ圧を示すゲージ（中央の棒） | 画面中央 |
| 顔選択UI | 漢/嘲/怒/笑/凄の5選択肢 | 画面下部 |
| テキスト | 相手のリアクションセリフ | 画面上部 |
| 効果音指示 | SE再生タイミング制御 | — |

### 操作方法
| 入力 | 処理 |
|---|---|
| 十字キー左右 | 顔の種類を選択 |
| ○ボタン | 選択した顔でメンチ確定・ケンカ申込み |
| ×ボタン | メンチをやめてフィールドに戻る |
| L+R同時押し | シャミセン発動（特殊タンカモード） |

### 入力受付
- メンチアニメーション再生中（約1.5秒）は入力無効
- 相手の反応テキスト表示中は○入力のみ受付（早送り）

---

## 5. タンカ画面（SCR_TANKA）

### 画面概要
口頭での挑発バトル。セリフ断片を正しい順序・タイミングで入力してタンカを完成させるリズムゲーム的画面。

### 表示要素
| 要素 | 内容 | 位置 |
|---|---|---|
| セリフ断片アイコン | ○△□×のボタンアイコン付きセリフ断片が上から落下 | 画面上部〜中央 |
| 判定ライン | 入力タイミングを示す横線 | 画面中央 |
| 判定テキスト | PERFECT / GREAT / GOOD / BAD / MISS | 判定ライン上 |
| タンカ完成テキスト | 入力済みセリフが左上に積み重なって表示 | 画面左上 |
| タイムバー | 制限時間を示す減少バー | 画面最上部 |
| コンボ数 | 連続成功数 | 画面右上 |
| キャラクター立ち絵 | 両者の立ち絵（表情変化あり） | 画面左右端 |

### 操作方法
| 入力 | 処理 |
|---|---|
| ○△□×ボタン | 落下中のアイコンに対応する入力 |
| L+R同時押し | シャミセン発動（単語選択でセリフ完成） |

### 入力受付
- タイムバーが0になると強制終了・タンカ敗北
- 全セリフ断片の入力完了で成功

---

## 6. 戦闘画面（SCR_BATTLE）

### 画面概要
タンカ後の喧嘩バトル本編。画面上部から落下するアイコンと同じボタンをタイミングよく入力する音ゲー的バトル画面。

### 表示要素
| 要素 | 内容 | 位置 |
|---|---|---|
| 主人公3Dモデル | バトル中の主人公キャラクター | 画面左中央 |
| 敵3Dモデル | 敵キャラクター（1〜5体） | 画面右中央 |
| 攻撃アイコン | ○△□×/L/Rのボタンアイコンが上から落下 | 画面上部 |
| 判定ライン | 入力タイミングの基準線 | 画面中央横ライン |
| 主人公HPバー | 主人公HP（赤バー） | 画面左下 |
| 敵HPバー | 敵HP（緑バー） | 画面右上 |
| 気合ゲージ | 超気合技ゲージ | 主人公HPバー下 |
| ブチギレゲージ | 蓄積で自動発動 | 画面左端縦バー |
| コンボ数 | 連続攻撃成功数 | 画面中央上 |
| BGM | 戦闘テーマ曲 | — |

### 操作方法
| 入力 | 処理 |
|---|---|
| ○△□×ボタン | 落下アイコンに合わせた攻撃入力 |
| Lボタン長押し | ガード（ダメージ軽減） |
| 十字キー（攻撃中以外） | ステップ回避 |
| L+R+○（気合MAX時） | 超気合技発動 |
| SELECT | 仲間への指示コマンド（集団喧嘩時） |

### 入力受付
- ダウン・ノックバックアニメーション中は入力無効（約0.8秒）
- 超気合技発動入力は気合ゲージMAX時のみ受付

---

## 7. ステータス画面（SCR_STATUS）

### 画面概要
主人公の現在のパラメータ・習得技・男気状態・ダチ一覧などを確認する情報画面。戦闘中も確認可能（ポーズ扱い）。

### 表示要素
| 要素 | 内容 | 位置 |
|---|---|---|
| キャラクター立ち絵 | 現在の外見（カスタマイズ反映） | 画面左側 |
| パラメータ表 | HP/気合/攻撃/防御/スピードの数値 | 画面右上 |
| 男気ゲージ表示 | ゲージ・顔アイコン・称号テキスト | 画面右中央 |
| 技一覧 | 習得済み技リスト（タブ切替） | 画面下部 |
| ダチ一覧 | 登録済みの友人リスト（最大100名以上） | タブ切替 |
| 武勇伝リスト | 達成済み武勇伝の一覧 | タブ切替 |
| プレイ時間 | 累計プレイ時間（時間：分） | 画面右下隅 |

### 操作方法
| 入力 | 処理 |
|---|---|
| LR各ボタン | 表示タブ切替（パラメータ/技/ダチ/武勇伝） |
| 十字キー | リスト内カーソル移動 |
| ○ボタン | 詳細表示（技・ダチの詳細情報） |
| ×ボタン / STARTボタン | ステータス画面を閉じてフィールドに戻る |

### 入力受付
- ポーズ中のため時間の進行は停止
- ダチ一覧の詳細確認中は○/×のみ受付
