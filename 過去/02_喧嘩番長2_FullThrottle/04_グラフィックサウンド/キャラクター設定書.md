## 武田トモヤ（主人公）

**年齢・所属・役割**
- 年齢: 17歳（高校2年生）
- 所属: 極東高校 2年B組
- 役割: プレイヤー操作キャラクター・本作の主人公

**外見**
- 身長: 178cm / 体重: 68kg
- 体型: やや細身だが均整の取れた体格。鍛えているが大柄ではない
- 髪型・色: 黒髪を後ろに軽く流したセミリーゼント。不精ヒゲなし
- 衣装（初期）: 極東高校の学ランに白いインナー。ボンタンは中学時代から愛用の青色
- 衣装（カスタマイズ）: 上半身インナー・アウターを自由に変更可能。特攻服・ジャージ・Tシャツなど多数

**性格・特徴・口癖**
- 「負け犬」時代の屈辱を内に秘めながらも、表面上はむしろ飄々としている
- 正直で裏表がなく、弱い者いじめを嫌う昭和気質
- 口癖（極東連合編）: 「俺には俺のやり方がある」「男が一度決めたことは曲げん」
- 口癖（東関狂走連合編）: 「ついてこれないなら退け」「遠慮は美徳じゃない」
- ルートによって性格が大きく変わることが本作の面白さ

**背景・経歴**
- 小学校から幼馴染の加藤葉子と同じクラスを続ける
- 中学2年で「二中の狂犬」として地域に名を馳せ、無敗の強者として君臨
- 高校入学の春、田中ヤスオに一蹴されて「ポチ」と呼ばれるようになる
- 如月亮は中学時代の最大の友人かつライバル

**得意技**
- 初期: ヘナチョコジャブ（弱い一撃）
- 成長後: 憤怒の炎・覇天光・熱き血潮（道場習得）

- 役割: 序盤の壁・極東連合編での兄貴分・脇役主人公

**外見**

- 体型: がっしりとした筋肉質。肩幅が広く威圧感がある
- 髪型・色: 黒髪の後退したリーゼント。短くまとめた実戦的なスタイル
- 衣装: 紺色の特攻服（背中に「極東連合」の刺繍）。インナーは白の長袖シャツ

- 昭和気質の正統派番長。義理・人情・礼節を重んじる
- 弱い者・女性・一般市民への暴力を厳しく禁じる
- 口癖: 「筋を通せ」「情けが男を作る」「力は守るためにある」

- 卒業を控えながら、後継者探しという個人的な課題も抱えている
- 如月亮とは学校の異なる好敵手として昔から互いを知っている

**得意技**

**年齢・所属・役割**
- 年齢: 17歳
- 所属: 東関狂走連合 / 暗黒騎士団 総長
- 役割: 極東連合編ラストボス / 東関狂走連合編の相棒・事実上の共同主人公

**外見**
- 身長: 179cm / 体重: 65kg
- 体型: 細身だが驚異的な俊敏さを持つ。スピード系の体格
- 髪型・色: 白髪（ウェーブがかかった長めの前髪）。染色ではなく天然の白髪という設定
- 衣装: 白色の特攻服（背中に「暗黒騎士団」の紋章）。全身白でまとめる独特のスタイル

**性格・特徴・口癖**
- 冷静で知略家。感情を表に出さず、常に一手先を読んで動く
- かつての温かい性格は10歳の時の喪失体験で仮面に閉じ込められた
- トモヤに対してのみ、わずかに素の表情を見せる
- 口癖: 「感傷は要らない」「動機があれば何でも正義になる」「友達ってのは厄介だな」

**背景・経歴**
- 中学2年でトモヤと「二中最強タッグ」として無敗を誇った
- 10歳の時、育ての親（祖父代わりの老人）が竜心会の若頭・日向誠に因縁をつけられ死亡
- 復讐を誓い、高校には進学せず組織づくりに専念
- 2年間で東関狂走連合を一代で築き上げた

**得意技**
- 超必殺技「国士無双」: 目にも留まらぬ連続蹴り技。発動から0.3秒以内に相手を囲む全方位攻撃

- 所属: 竜心会 若頭

- 髪型・色: 黒髪を短く刈り込んだオールバック
