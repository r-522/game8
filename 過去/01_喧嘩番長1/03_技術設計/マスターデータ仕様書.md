# マスターデータ仕様書 — 喧嘩番長

## 技マスターテーブル

### テーブル定義

| カラム名 | 型 | サイズ | 必須 | 説明 |
|---------|------|------|------|------|
| id | int | 4 | YES | 技ID（主キー） |
| name | string | 32 | YES | 技名 |
| type | enum | 1 | YES | タイプ（NORMAL/SPECIAL/SUPER/THROW/DASH） |
| power | int | 4 | YES | 威力係数（基礎ダメージへの乗算係数 × 10、例: 15 = ×1.5） |
| kiAiCost | int | 4 | YES | 消費気合（0の場合は通常技） |
| unlockLevel | int | 4 | YES | 習得可能番長度 |
| animId | int | 4 | YES | アニメーションID（参照） |
| hitboxType | enum | 1 | YES | ヒットボックス形状（FRONT/AREA/LAUNCH） |
| description | string | 128 | NO | 技説明文 |
